# FT-L01-17-collaboration-commentaires

Cette fiche technique explique comment collaborer à plusieurs sur un document Word et utiliser les commentaires pour communiquer.

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                      |
|-----------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Collaborer et commenter dans Word                                                                                                                                                                                               |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                              |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Ajouter un commentaire <br/>• Répondre à un commentaire <br/>• Partager un document avec d’autres utilisateurs                                                                   |
| **Définitions clés**                          | **Commentaire** : Note ajoutée dans la marge pour donner une indication ou poser une question. <br/>**Partage** : Permettre à d’autres personnes d’accéder au document.                                                          |
| **Explication de la notion avec exemples**    | L’élève sélectionne un mot et insère un commentaire pour demander une correction. Il partage ensuite le document avec un camarade pour qu’il puisse le modifier.                                                                |
| **Méthodologie ou étapes à suivre**           | 1. Sélectionner le texte <br/>2. Aller dans l’onglet **Révision** <br/>3. Cliquer sur **Nouveau commentaire** <br/>4. Taper le message <br/>5. Cliquer sur **Partager** pour inviter d’autres personnes                           |
| **Illustration ou schéma explicatif**         | Capture d’écran montrant un commentaire dans la marge et le bouton **Partager**.                                                                                                                                                |
| **Exercice d’application ou question**        | Demander à l’élève d’insérer un commentaire sur une phrase, puis de partager le document avec un camarade.                                                                                                                      |
| **Mot-clé ou astuce mnémotechnique**          | **COPA** : <br/>**CO**mmenter <br/>**P**artager <br/>**A**viser                                                                                                                                                                 |
