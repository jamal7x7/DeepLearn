# FT-L01-19-raccourcis-clavier

Cette fiche technique présente les principaux raccourcis clavier pour gagner du temps dans Microsoft Word.

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                      |
|-----------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Raccourcis clavier dans Word                                                                                                                                                                                                    |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                              |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Copier, couper, coller avec le clavier <br/>• Annuler une action <br/>• Sauvegarder rapidement                                                                                  |
| **Définitions clés**                          | **Raccourci clavier** : Combinaison de touches permettant d’effectuer une action rapidement sans utiliser la souris.                                                                                                            |
| **Explication de la notion avec exemples**    | Pour copier un texte, l’élève sélectionne le texte et appuie sur **Ctrl + C**. Pour coller, il utilise **Ctrl + V**. Pour sauvegarder, **Ctrl + S**.                                                                            |
| **Méthodologie ou étapes à suivre**           | 1. Sélectionner le texte <br/>2. Appuyer sur **Ctrl + C** pour copier <br/>3. Placer le curseur à l’endroit souhaité <br/>4. Appuyer sur **Ctrl + V** pour coller <br/>5. Utiliser **Ctrl + Z** pour annuler une action         |
| **Illustration ou schéma explicatif**         | Tableau des raccourcis : <br/>• **Ctrl + C** : Copier <br/>• **Ctrl + X** : Couper <br/>• **Ctrl + V** : Coller <br/>• **Ctrl + Z** : Annuler <br/>• **Ctrl + S** : Sauvegarder                                                 |
| **Exercice d’application ou question**        | Demander à l’élève de copier une phrase, de la coller ailleurs, puis d’annuler l’action.                                                                                                                                        |
| **Mot-clé ou astuce mnémotechnique**          | **CAPS** : <br/>**C**opier <br/>**A**nnuler <br/>**P**aste (coller) <br/>**S**auvegarder                                                                                                                                       |
