# FT-SI05 - Connectivité

Cette fiche technique explique comment les ordinateurs se connectent entre eux et à Internet.

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                      |
|-----------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | La connectivité dans un système informatique                                                                                                                                                                                    |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                              |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Comprendre ce qu’est un réseau <br/>• Différencier connexion filaire et sans fil <br/>• Connaître les usages d’Internet                                                         |
| **Définitions clés**                          | **Réseau** : Ensemble d’ordinateurs reliés entre eux. <br/>**Wi-Fi** : Connexion sans fil. <br/>**Internet** : Réseau mondial reliant des millions d’ordinateurs.                                                                |
| **Explication de la notion avec exemples**    | À l’école, les ordinateurs sont connectés en réseau pour partager une imprimante. À la maison, on utilise le Wi-Fi pour accéder à Internet.                                                                                     |
| **Méthodologie ou étapes à suivre**           | 1. Identifier les appareils connectés <br/>2. Distinguer filaire (câble) et sans fil (Wi-Fi) <br/>3. Expliquer comment accéder à Internet                                                                                       |
| **Illustration ou schéma explicatif**         | Schéma montrant plusieurs ordinateurs reliés par câble et par Wi-Fi à une box Internet.                                                                                                                                        |
| **Exercice d’application ou question**        | Demander à l’élève de dessiner un réseau domestique avec box, ordinateurs, smartphones.                                                                                                                                        |
| **Mot-clé ou astuce mnémotechnique**          | **RWI** : <br/>**R**éseau <br/>**W**i-Fi <br/>**I**nternet                                                                                                                                                                     |
