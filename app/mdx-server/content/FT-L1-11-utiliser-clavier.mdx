# FT-L1-11-utiliser-clavier
Voici une fiche technique sous forme de tableau, conçue pour des élèves de 7e année (niveau 1), qui aborde la notion d'utilisation efficace du clavier dans Microsoft Word 2021 :

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Utiliser le Clavier Efficacement dans Microsoft Word 2021                                                                                                                                                                                                                                                                                                                                                                                         |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Utiliser les touches de fonction <br/>• Maîtriser les raccourcis clavier essentiels <br/>• Naviguer efficacement dans un document <br/>• Saisir des caractères spéciaux français                                                                                                                                                                                                                |
| **Définitions clés**                          | **Touches de fonction** : Touches F1 à F12 en haut du clavier qui exécutent des commandes spécifiques. <br/>**Raccourci clavier** : Combinaison de touches permettant d'exécuter rapidement une action sans utiliser la souris. <br/>**Caractères spéciaux** : Lettres accentuées, symboles et autres caractères qui ne sont pas directement accessibles sur le clavier.                                                                                                                                                                          |
| **Explication de la notion avec exemples**    | L'élève utilise les raccourcis clavier pour gagner du temps et être plus efficace. Par exemple, au lieu de cliquer sur l'icône "Enregistrer" ou d'aller dans le menu Fichier, il peut simplement appuyer sur Ctrl+S. Pour copier et coller du texte, il utilise Ctrl+C et Ctrl+V au lieu de passer par le menu contextuel, ce qui accélère considérablement son travail.                                                                                                                       |
| **Méthodologie ou étapes à suivre**           | 1. **Apprendre les touches de fonction** : F1 pour l'aide, F5 pour la boîte de dialogue "Atteindre", F7 pour la vérification orthographique, F12 pour "Enregistrer sous".<br/>2. **Maîtriser les raccourcis essentiels** : Ctrl+C (copier), Ctrl+X (couper), Ctrl+V (coller), Ctrl+Z (annuler), Ctrl+S (enregistrer), Ctrl+B (gras), Ctrl+I (italique), Ctrl+U (souligné).<br/>3. **Naviguer dans le document** : Ctrl+Début (aller au début du document), Ctrl+Fin (aller à la fin), Ctrl+Flèches (se déplacer par mot ou paragraphe).<br/>4. **Saisir des caractères spéciaux** : ^ puis e pour ê, ' puis e pour é, ` puis e pour è, , puis c pour ç. |
| **Illustration ou schéma explicatif**         | **Schéma des raccourcis clavier** : <br/>• Les touches **Ctrl+C, Ctrl+X, Ctrl+V** pour copier, couper, coller. <br/>• Les touches **Ctrl+B, Ctrl+I, Ctrl+U** pour le gras, l'italique, le souligné. <br/>• Les touches **Ctrl+Z, Ctrl+Y** pour annuler et rétablir. <br/>• Les combinaisons pour les **caractères accentués** français.                                                                                   |
| **Exercice d'application ou question**        | **Exercice** : Demander à l'élève de réaliser les actions suivantes en utilisant uniquement le clavier : <br/>1. Créer un nouveau document (Ctrl+N) <br/>2. Écrire un court paragraphe avec des lettres accentuées <br/>3. Mettre certains mots en gras, en italique et en souligné avec les raccourcis <br/>4. Enregistrer le document (Ctrl+S) puis le fermer (Ctrl+W)                                                                                                                    |
| **Mot-clé ou astuce mnémotechnique**          | **CLAVIER** : <br/>**C**opier avec Ctrl+C – **L**ier avec Ctrl+V – **A**nnuler avec Ctrl+Z – **V**érifier avec F7 – **I**talique avec Ctrl+I – **E**nregistrer avec Ctrl+S – **R**accourcis pour gagner du temps                                                                                                                                                                                                                                                                                                                           |
