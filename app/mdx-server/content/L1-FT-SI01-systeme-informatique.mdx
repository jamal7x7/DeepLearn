# FT-SI01 - Système informatique

Cette fiche technique présente l’architecture matérielle d’un système informatique.

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                      |
|-----------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Architecture d’un système informatique                                                                                                                                                                                          |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                              |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Identifier les composants matériels <br/>• Comprendre le rôle du processeur, de la mémoire, des périphériques                                                                   |
| **Définitions clés**                          | **Unité centrale** : Boîtier contenant le processeur, la mémoire, la carte mère. <br/>**Périphériques** : Appareils connectés (clavier, souris, écran, imprimante).                                                              |
| **Explication de la notion avec exemples**    | L’ordinateur est composé d’une unité centrale reliée à des périphériques. Par exemple, la souris permet de déplacer le curseur, l’imprimante d’obtenir une copie papier.                                                        |
| **Méthodologie ou étapes à suivre**           | 1. Observer un ordinateur <br/>2. Nommer chaque composant <br/>3. Relier chaque périphérique à sa fonction                                                                                                                      |
| **Illustration ou schéma explicatif**         | Schéma d’un ordinateur avec flèches indiquant le processeur, la mémoire, les périphériques.                                                                                                                                     |
| **Exercice d’application ou question**        | Demander à l’élève de dessiner un ordinateur et de nommer ses composants.                                                                                                                                                       |
| **Mot-clé ou astuce mnémotechnique**          | **CPU** : <br/>**C**erveau (processeur) <br/>**P**ériphériques <br/>**U**nité centrale                                                                                                                                          |
