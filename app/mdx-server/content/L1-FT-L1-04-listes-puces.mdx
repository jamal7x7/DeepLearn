# FT-L1-04-listes-puces
Voici une fiche technique sous forme de tableau, conçue pour des élèves de 7e année (niveau 1), qui aborde la notion de création et de gestion des listes à puces et numérotées dans Microsoft Word 2021 :

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Créer des Listes dans Microsoft Word 2021                                                                                                                                                                                                                                                                                                                                                                                         |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Créer une liste à puces <br/>• Créer une liste numérotée <br/>• Modifier une liste existante <br/>• Créer des listes imbriquées (à plusieurs niveaux)                                                                                                                                                                                                                |
| **Définitions clés**                          | **Liste à puces** : Liste d'éléments précédés de symboles (points, tirets, etc.) pour organiser l'information. <br/>**Liste numérotée** : Liste d'éléments précédés de numéros ou lettres, utile pour les étapes séquentielles. <br/>**Liste imbriquée** : Liste à plusieurs niveaux, où des sous-éléments sont indentés sous des éléments principaux.                                                                                                                                                                          |
| **Explication de la notion avec exemples**    | L'élève utilise les listes pour organiser clairement ses idées. Par exemple, pour présenter les ingrédients d'une recette (liste à puces) ou les étapes à suivre dans un processus (liste numérotée). Les listes imbriquées permettent de hiérarchiser l'information, comme pour un plan de devoir avec des parties et sous-parties.                                                                                                                       |
| **Méthodologie ou étapes à suivre**           | 1. **Créer une liste à puces** : Écrire les éléments les uns sous les autres, les sélectionner, puis cliquer sur l'icône "Puces" dans l'onglet "Accueil".<br/>2. **Créer une liste numérotée** : Sélectionner les éléments et cliquer sur l'icône "Numérotation" dans l'onglet "Accueil".<br/>3. **Modifier une liste** : Ajouter un nouvel élément en appuyant sur Entrée à la fin d'un élément. Pour terminer la liste, appuyer deux fois sur Entrée.<br/>4. **Créer une liste imbriquée** : Dans une liste existante, placer le curseur au début d'un élément et appuyer sur Tab pour l'indenter (créer un sous-niveau). |
| **Illustration ou schéma explicatif**         | **Schéma des options de listes dans Word** : <br/>• L'onglet **Accueil** contient les boutons "Puces" et "Numérotation". <br/>• Le bouton **Puces** affiche différents styles de puces disponibles. <br/>• Le bouton **Numérotation** montre les différents formats de numérotation (1, 2, 3 ou a, b, c, etc.).                                                                                   |
| **Exercice d'application ou question**        | **Exercice** : Demander à l'élève de créer un document sur ses activités du week-end avec : <br/>1. Une liste à puces pour les activités du samedi <br/>2. Une liste numérotée pour les activités du dimanche <br/>3. Une liste imbriquée pour ajouter des détails à certaines activités                                                                                                                    |
| **Mot-clé ou astuce mnémotechnique**          | **LISTE** : <br/>**L**ister les éléments – **I**nsérer les puces ou numéros – **S**tructure hiérarchique (imbrication) – **T**erminer par double Entrée – **E**nrichir avec différents styles de puces                                                                                                                                                                                                                                                                                                                           |
