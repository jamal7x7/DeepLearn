# FT-L1-13-caracteres-speciaux-symboles
Voici une fiche technique sous forme de tableau, conçue pour des élèves de 7e année (niveau 1), qui aborde la notion d'insertion de caractères spéciaux et symboles dans Microsoft Word 2021 :

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Caractères Spéciaux et Symboles dans Microsoft Word 2021                                                                                                                                                                                                                                                                                                                                                                                         |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Insérer des symboles courants (©, ®, €, etc.) <br/>• Utiliser les caractères spéciaux mathématiques <br/>• Insérer des fractions et exposants <br/>• Créer des équations simples                                                                                                                                                                                                                |
| **Définitions clés**                          | **Caractère spécial** : Lettre, chiffre ou symbole qui n'est pas directement accessible sur le clavier standard. <br/>**Symbole** : Signe graphique représentant une idée, une opération ou une entité. <br/>**Équation** : Expression mathématique pouvant contenir des symboles spéciaux. <br/>**Exposant/Indice** : Caractère placé légèrement au-dessus/en-dessous de la ligne de texte.                                                                                                                                                                          |
| **Explication de la notion avec exemples**    | L'élève utilise les caractères spéciaux et symboles pour enrichir ses documents et exprimer des notions spécifiques. Par exemple, dans un document sur les mathématiques, il peut insérer le symbole π pour représenter pi, utiliser des exposants pour écrire des formules comme E=mc², ou insérer des fractions comme ½ au lieu d'écrire 1/2. Dans un texte sur le commerce, il peut utiliser le symbole € pour les prix ou © pour le copyright.                                                                                                                       |
| **Méthodologie ou étapes à suivre**           | 1. **Insérer des symboles courants** : Aller dans l'onglet "Insertion", cliquer sur "Symbole", choisir parmi les symboles récemment utilisés ou cliquer sur "Autres symboles" pour plus d'options.<br/>2. **Utiliser les caractères mathématiques** : Dans la boîte de dialogue "Symbole", sélectionner la police "Symbol" pour accéder aux caractères grecs et mathématiques comme π, Σ, ∞.<br/>3. **Insérer des fractions et exposants** : Pour les fractions, utiliser la boîte de dialogue "Symbole". Pour les exposants, sélectionner le caractère et utiliser le format "Exposant" dans la boîte de dialogue "Police" (Ctrl+Maj+F).<br/>4. **Créer des équations** : Aller dans l'onglet "Insertion", cliquer sur "Équation" et utiliser les outils de l'éditeur d'équations pour créer des formules mathématiques. |
| **Illustration ou schéma explicatif**         | **Schéma des options de symboles** : <br/>• L'onglet **Insertion** contient le bouton "Symbole". <br/>• La boîte de dialogue **Symbole** affiche les caractères disponibles par police. <br/>• Le menu **Équation** dans l'onglet Insertion permet d'accéder à l'éditeur d'équations. <br/>• Les options de **mise en forme** permettent de créer des exposants et indices.                                                                                   |
| **Exercice d'application ou question**        | **Exercice** : Demander à l'élève de créer un document de mathématiques contenant : <br/>1. Des fractions (½, ¼) <br/>2. Le symbole π correctement inséré <br/>3. Une formule avec exposant comme A = πr² <br/>4. Des symboles de comparaison (≤, ≥, ≠) <br/>5. Une équation simple comme E = mc²                                                                                                                    |
| **Mot-clé ou astuce mnémotechnique**          | **SYMBOL** : <br/>**S**électionner l'emplacement – **Y**eux sur l'onglet Insertion – **M**enu Symbole – **B**oîte de dialogue – **O**ptions spéciales (fractions, exposants) – **L**iste des symboles récents                                                                                                                                                                                                                                                                                                                           |
