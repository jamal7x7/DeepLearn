# L'Information

<ExerciseCard title="Découvrir le concept d'information">

Dans cette leçon, tu vas découvrir ce qu'est l'information, comment elle est représentée dans un système informatique et pourquoi elle est si importante dans notre monde numérique.

Voici ce que tu vas apprendre :
- Ce qu'est l'information et comment elle diffère des données
- Comment l'information est codée et stockée numériquement
- Les différentes unités de mesure de l'information
- Comment gérer efficacement l'information

</ExerciseCard>

## Exercice 1: Conversion en binaire

Convertis les caractères suivants en code binaire (utilise la table ASCII) :
1. La lettre "M"
2. Le chiffre "5"
3. Le symbole "&"

<details>
  <summary>Solution</summary>
  <p>1. M = ASCII 77 = 01001101</p>
  <p>2. 5 = ASCII 53 = 00110101</p>
  <p>3. & = ASCII 38 = 00100110</p>
</details>

## Exercice 2: Calcul de taille de fichiers

Calcule la taille approximative des fichiers suivants :
1. Un document texte de 2000 caractères (1 caractère = 1 octet)
2. Une image de 1920×1080 pixels où chaque pixel est codé sur 3 octets (RGB)
3. Un enregistrement audio de 3 minutes échantillonné à 44,1 kHz en stéréo, avec 16 bits par échantillon

<details>
  <summary>Solution</summary>
  <p>1. Document texte : 2000 caractères × 1 octet = 2000 octets = 2 Ko</p>
  <p>2. Image : 1920 × 1080 pixels × 3 octets = 6 220 800 octets ≈ 6 Mo</p>
  <p>3. Audio : 3 minutes × 60 secondes × 44100 échantillons/seconde × 2 canaux × 2 octets = 31 752 000 octets ≈ 30 Mo</p>
</details>

## Exercice 3: Identification des formats

Pour chaque type d'information, indique le format de fichier le plus approprié :
1. Un document texte avec mise en forme complexe destiné à être imprimé
2. Une photo de haute qualité pour un site web (doit être légère)
3. Un logo d'entreprise avec des formes géométriques qui doit pouvoir être redimensionné sans perte de qualité

<details>
  <summary>Solution</summary>
  <p>1. Format PDF (.pdf) - préserve la mise en forme exacte pour l'impression</p>
  <p>2. Format JPEG (.jpg) - bon compromis entre qualité et taille pour les photos sur le web</p>
  <p>3. Format SVG (.svg) - format vectoriel qui permet un redimensionnement sans perte de qualité</p>
</details>

## Défis créatifs

1. **Créer une infographie** : Réalise une infographie qui explique les différentes unités de mesure de l'information et donne des exemples concrets pour chacune.

2. **Analyser ton usage numérique** : Pendant une journée, note tous les types d'informations numériques que tu utilises (textes, images, vidéos, etc.) et estime la quantité totale d'information traitée.

3. **Comparer des formats** : Prends une même photo et enregistre-la dans différents formats (PNG, JPEG avec différents niveaux de compression, etc.). Compare les tailles de fichiers et la qualité visuelle.
