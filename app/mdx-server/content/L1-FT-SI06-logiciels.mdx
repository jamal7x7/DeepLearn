# FT-SI06 - Logiciels

Cette fiche technique explique ce qu’est un logiciel et les différents types de logiciels.

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                      |
|-----------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Les logiciels dans un système informatique                                                                                                                                                                                      |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                              |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Définir un logiciel <br/>• Différencier système d’exploitation et application <br/>• Donner des exemples de logiciels                                                            |
| **Définitions clés**                          | **Logiciel** : Programme qui permet à l’ordinateur d’effectuer des tâches. <br/>**Système d’exploitation** : Logiciel qui gère l’ordinateur (Windows, macOS). <br/>**Application** : Logiciel pour une tâche précise (Word, jeux). |
| **Explication de la notion avec exemples**    | Windows est un système d’exploitation. Word est une application pour écrire. Minecraft est un jeu vidéo, donc une application.                                                                                                  |
| **Méthodologie ou étapes à suivre**           | 1. Lister les logiciels connus <br/>2. Classer en système ou application <br/>3. Associer chaque logiciel à sa fonction                                                                                                         |
| **Illustration ou schéma explicatif**         | Schéma séparant système d’exploitation et applications avec exemples.                                                                                                                                                           |
| **Exercice d’application ou question**        | Demander à l’élève de classer une liste de logiciels dans la bonne catégorie.                                                                                                                                                   |
| **Mot-clé ou astuce mnémotechnique**          | **SAL** : <br/>**S**ystème <br/>**A**pplication <br/>**L**ogiciel                                                                                                                                                               |
