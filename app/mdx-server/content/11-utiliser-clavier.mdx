# Utiliser le Clavier et Caractères Spéciaux

<ExerciseCard title="Exercice sur les caractères spéciaux">

Voici un texte à compléter avec des caractères spéciaux:

```
Ma société s'appelle "Super Créations".
Nous vendons nos produits à 15€ l'unité.
La température idéale pour nos produits est de 25° C.
Notre logo est protégé par copyright © 2023.
Notre nom est une marque déposée ®.
```

Essaie de:
1. Remplacer les guillemets droits par des guillemets français « »
2. Insérer le symbole € correctement
3. Ajouter le symbole ° pour les degrés
4. Insérer les symboles © et ®

</ExerciseCard>

        <li>Ctrl+X: Couper le texte sélectionné</li>
        <li>Ctrl+V: Coller le texte</li>
        <li>Ctrl+Z: Annuler la dernière action</li>
        <li>Ctrl+Y: Rétablir la dernière action annulée</li>
      </ol>
    )
  },
  {
    title: "Étape 3: Insérer des caractères spéciaux",
    content: (
      <ol>
        <li>Va dans l'onglet "Insertion"</li>
        <li>Clique sur "Symbole" puis "Autres symboles"</li>
        <li>Choisis un symbole comme © ou ®</li>
        <li>Clique sur "Insérer"</li>
      </ol>
    )
  },
  {
    title: "Étape 4: Utiliser les raccourcis pour les caractères",
    content: (
      <ol>
        <li>Alt+0169: ©</li>
        <li>Alt+0174: ®</li>
        <li>Alt+0176: ° (degré)</li>
        <li>Alt+0128: € (euro)</li>
      </ol>
    )
  }
]} />

<Tip>
Pour les raccourcis Alt, utilise le pavé numérique en maintenant Alt enfoncé!
</Tip>
