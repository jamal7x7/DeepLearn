# FT-L08-references-absolues-relatives
Voici une fiche technique sous forme de tableau, conçue pour des élèves de 7e année (niveau 1), qui aborde la notion de références absolues et relatives dans Microsoft Excel 2021 :

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Références Absolues et Relatives dans Excel 2021                                                                                                                                                                                                                                                                                                                                                                                                         |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Comprendre la différence entre références relatives et absolues <br/>• Utiliser le symbole $ pour créer des références absolues <br/>• Appliquer des références mixtes (ligne ou colonne fixe) <br/>• Copier des formules en utilisant les références appropriées                                                                                                                                                                                                                |
| **Définitions clés**                          | **Référence relative** : Adresse de cellule qui change automatiquement lorsque la formule est copiée (ex: A1). <br/>**Référence absolue** : Adresse de cellule qui reste fixe lorsque la formule est copiée (ex: $A$1). <br/>**Référence mixte** : Adresse où soit la ligne, soit la colonne reste fixe (ex: $A1 ou A$1). <br/>**Symbole $** : Caractère utilisé pour indiquer qu'une partie de la référence doit rester fixe.                                                                                                                                                                          |
| **Explication de la notion avec exemples**    | L'élève crée un tableau de conversion d'euros en différentes devises. Dans la cellule B2, il saisit le taux de change euro/dollar. Pour convertir un montant en euros (colonne A) en dollars (colonne C), il utilise la formule =A4*B2. S'il copie cette formule vers le bas pour d'autres montants, la référence à A4 devient A5, A6, etc. (référence relative), ce qui est correct. Mais la référence à B2 devient B3, B4, etc., ce qui est incorrect car le taux de change reste en B2. En utilisant la formule =A4*$B$2 (référence absolue pour le taux), il peut copier la formule et le taux reste fixe.                                                                                                                       |
| **Méthodologie ou étapes à suivre**           | 1. **Comprendre les références relatives** : <br/>   - Par défaut, les références de cellules sont relatives (A1, B2, etc.) <br/>   - Lorsqu'une formule est copiée, ces références s'ajustent en fonction de la position relative<br/>2. **Créer une référence absolue** : <br/>   - Pour fixer une référence, placer le symbole $ devant la lettre de colonne et le numéro de ligne ($A$1) <br/>   - Utiliser la touche F4 pour faire défiler les différents types de références (A1 → $A$1 → A$1 → $A1)<br/>3. **Utiliser des références mixtes** : <br/>   - $A1 : la colonne A est fixe, mais la ligne change <br/>   - A$1 : la ligne 1 est fixe, mais la colonne change<br/>4. **Appliquer dans des formules** : <br/>   - Identifier les références qui doivent rester fixes (taux, constantes) <br/>   - Utiliser des références absolues pour ces valeurs <br/>   - Garder des références relatives pour les valeurs qui doivent changer<br/>5. **Copier les formules** : <br/>   - Utiliser la poignée de recopie pour dupliquer la formule <br/>   - Vérifier que les références se comportent comme prévu |
| **Illustration ou schéma explicatif**         | **Schéma des types de références** : <br/>• **A1** (référence relative) : La colonne et la ligne changent lors de la copie. <br/>• **$A$1** (référence absolue) : Ni la colonne ni la ligne ne changent lors de la copie. <br/>• **$A1** (référence mixte) : La colonne reste fixe, la ligne change. <br/>• **A$1** (référence mixte) : La ligne reste fixe, la colonne change. <br/>• La touche **F4** permet de basculer entre ces différents types de références.                                                                                   |
| **Exercice d'application ou question**        | **Exercice** : Demander à l'élève de créer un tableau de conversion de températures : <br/>1. Dans la cellule A1, saisir "Celsius" et dans B1, saisir "Fahrenheit" <br/>2. Dans la colonne A (A2:A10), saisir différentes températures en Celsius <br/>3. Dans la cellule B2, créer une formule pour convertir Celsius en Fahrenheit : =(A2*9/5)+32 <br/>4. Copier cette formule vers le bas (B3:B10) <br/>5. Créer un second tableau avec un facteur de conversion unique : <br/>   - Dans D1, saisir "Facteur" et dans E1 saisir 9/5 <br/>   - Dans D2, saisir "Addition" et dans E2 saisir 32 <br/>   - Dans A12, saisir "Celsius" et dans B12, saisir "Fahrenheit" <br/>   - Dans la colonne A (A13:A20), saisir différentes températures en Celsius <br/>   - Dans la cellule B13, créer une formule utilisant les références absolues : =(A13*$E$1)+$E$2 <br/>   - Copier cette formule vers le bas (B14:B20) <br/>6. Comparer les résultats des deux méthodes                                                                                                                    |
| **Mot-clé ou astuce mnémotechnique**          | **DOLLAR** : <br/>**D**éfinir ce qui doit rester fixe – **O**rganiser les références – **L**ier avec le symbole $ – **L**igne ou colonne fixe – **A**dresse absolue avec $A$1 – **R**ecopier les formules                                                                                                                                                                                                                                                                                                                           |
