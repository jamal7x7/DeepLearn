# FT-SI08 - Configuration monoposte

Cette fiche technique explique la configuration matérielle d’un ordinateur utilisé seul (monoposte).

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                      |
|-----------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Configuration d’un ordinateur monoposte                                                                                                                                                                                         |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                              |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Identifier les composants d’un ordinateur monoposte <br/>• Comprendre leur rôle                                                                                                 |
| **Définitions clés**                          | **Unité centrale** : Contient le processeur, la mémoire, le disque dur. <br/>**Périphériques** : Clavier, souris, écran, imprimante.                                                                                             |
| **Explication de la notion avec exemples**    | Un ordinateur personnel à la maison est un monoposte, utilisé seul, avec ses propres périphériques.                                                                                                                             |
| **Méthodologie ou étapes à suivre**           | 1. Lister les composants matériels <br/>2. Associer chaque composant à sa fonction <br/>3. Vérifier la connexion des périphériques                                                                                              |
| **Illustration ou schéma explicatif**         | Schéma d’un ordinateur monoposte avec ses périphériques.                                                                                                                                                                       |
| **Exercice d’application ou question**        | Demander à l’élève de nommer les composants d’un ordinateur monoposte.                                                                                                                                                         |
| **Mot-clé ou astuce mnémotechnique**          | **CPU-P** : <br/>**C**erveau (processeur) <br/>**P**ériphériques <br/>**U**nité centrale                                                                                                                                       |
