# FT-L04-fonctions-avancees
Voici une fiche technique sous forme de tableau, conçue pour des élèves de 7e année (niveau 1), qui aborde la notion de fonctions avancées dans Microsoft Excel 2021 :

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Fonctions Avancées dans Excel 2021                                                                                                                                                                                                                                                                                                                                                                                                         |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Utiliser la fonction SI pour créer des conditions <br/>• Appliquer les fonctions de comptage (NB, NBVAL, NB.SI) <br/>• Utiliser les fonctions de recherche (RECHERCHEV) <br/>• Combiner plusieurs fonctions dans une formule                                                                                                                                                                                                                |
| **Définitions clés**                          | **Fonction SI** : Fonction qui effectue un test logique et renvoie une valeur si le test est VRAI et une autre si le test est FAUX. <br/>**Fonction NB.SI** : Compte le nombre de cellules qui répondent à un critère spécifique. <br/>**Fonction RECHERCHEV** : Recherche une valeur dans la première colonne d'un tableau et renvoie une valeur dans la même ligne d'une colonne spécifiée. <br/>**Opérateurs de comparaison** : Symboles utilisés pour comparer des valeurs (=, >, <, >=, <=, <>).                                                                                                                                                                          |
| **Explication de la notion avec exemples**    | L'élève crée un tableau de résultats d'examens et utilise des fonctions avancées pour analyser les données. Par exemple, avec la fonction SI, il peut attribuer automatiquement une mention ("Admis" ou "Recalé") en fonction de la note obtenue : =SI(B2>=10;"Admis";"Recalé"). Avec NB.SI, il peut compter le nombre d'élèves ayant obtenu une note supérieure à 15 : =NB.SI(B2:B20;">15"). Avec RECHERCHEV, il peut associer automatiquement un barème de points à une note alphabétique en consultant un tableau de correspondance.                                                                                                                       |
| **Méthodologie ou étapes à suivre**           | 1. **Fonction SI** : <br/>   - Taper =SI( <br/>   - Définir la condition (ex: A1>10) <br/>   - Ajouter un point-virgule (;) <br/>   - Spécifier la valeur si VRAI <br/>   - Ajouter un autre point-virgule <br/>   - Spécifier la valeur si FAUX <br/>   - Fermer la parenthèse et valider<br/>2. **Fonctions de comptage** : <br/>   - Pour NB.SI, taper =NB.SI( <br/>   - Sélectionner la plage de cellules à analyser <br/>   - Ajouter un point-virgule <br/>   - Définir le critère (ex: ">10") <br/>   - Fermer la parenthèse et valider<br/>3. **Fonction RECHERCHEV** : <br/>   - Taper =RECHERCHEV( <br/>   - Spécifier la valeur à rechercher <br/>   - Ajouter un point-virgule <br/>   - Définir le tableau de recherche <br/>   - Ajouter un point-virgule <br/>   - Indiquer le numéro de la colonne contenant le résultat <br/>   - Ajouter un point-virgule <br/>   - Spécifier FAUX pour une correspondance exacte ou VRAI pour une correspondance approximative <br/>   - Fermer la parenthèse et valider |
| **Illustration ou schéma explicatif**         | **Schéma des fonctions avancées** : <br/>• La fonction **SI** se présente sous la forme =SI(test_logique;valeur_si_vrai;valeur_si_faux) <br/>• La fonction **NB.SI** se présente sous la forme =NB.SI(plage;critère) <br/>• La fonction **RECHERCHEV** se présente sous la forme =RECHERCHEV(valeur_recherchée;table_matrice;no_index_col;valeur_proche) <br/>• Les fonctions peuvent être **imbriquées** les unes dans les autres pour créer des formules plus complexes.                                                                                   |
| **Exercice d'application ou question**        | **Exercice** : Demander à l'élève de créer un tableau de suivi des notes d'une classe contenant : <br/>1. Les noms des élèves dans la colonne A <br/>2. Les notes sur 20 dans la colonne B <br/>3. Dans la colonne C, utiliser la fonction SI pour afficher "Admis" si la note est ≥ 10, sinon "Recalé" <br/>4. Dans la colonne D, utiliser une fonction SI imbriquée pour attribuer une mention : <br/>   - Si note ≥ 16 : "Très bien" <br/>   - Si note ≥ 14 : "Bien" <br/>   - Si note ≥ 12 : "Assez bien" <br/>   - Si note ≥ 10 : "Passable" <br/>   - Sinon : "Insuffisant" <br/>5. En dessous du tableau, utiliser NB.SI pour compter le nombre d'élèves admis <br/>6. Créer un petit tableau de correspondance entre les notes numériques et les lettres (A, B, C, D, E) et utiliser RECHERCHEV pour attribuer automatiquement une lettre à chaque note dans la colonne E                                                                                                                    |
| **Mot-clé ou astuce mnémotechnique**          | **SICNR** : <br/>**S**i pour les conditions – **I**mbriquer pour combiner – **C**ompter avec NB.SI – **N**aviguer dans les données – **R**echercher avec RECHERCHEV                                                                                                                                                                                                                                                                                                                           |
