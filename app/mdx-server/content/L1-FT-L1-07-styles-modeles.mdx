# FT-L1-07-styles-modeles
Voici une fiche technique sous forme de tableau, conçue pour des élèves de 7e année (niveau 1), qui aborde la notion d'utilisation des styles et modèles dans Microsoft Word 2021 :

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Utiliser les Styles et Modèles dans Microsoft Word 2021                                                                                                                                                                                                                                                                                                                                                                                         |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Appliquer un style prédéfini à un texte <br/>• Modifier un style existant <br/>• Utiliser un modèle de document <br/>• Créer un document avec une présentation professionnelle                                                                                                                                                                                                                |
| **Définitions clés**                          | **Style** : Ensemble de caractéristiques de mise en forme (police, taille, couleur, etc.) qu'on peut appliquer en un seul clic. <br/>**Modèle** : Document préformaté avec une mise en page, des styles et parfois du contenu prédéfinis. <br/>**Galerie de styles** : Collection de styles disponibles dans l'onglet Accueil.                                                                                                                                                                          |
| **Explication de la notion avec exemples**    | L'élève utilise les styles pour maintenir une présentation cohérente dans son document. Par exemple, en appliquant le style "Titre 1" à tous les titres principaux et "Titre 2" aux sous-titres, il crée une hiérarchie visuelle claire. Les modèles permettent de gagner du temps en partant d'une structure déjà formatée, comme pour un rapport ou une lettre.                                                                                                                       |
| **Méthodologie ou étapes à suivre**           | 1. **Appliquer un style** : Sélectionner le texte, puis dans l'onglet "Accueil", section "Styles", cliquer sur le style souhaité (ex: "Titre 1", "Titre 2").<br/>2. **Modifier un style** : Cliquer droit sur le style dans la galerie, choisir "Modifier", changer les paramètres (police, taille, couleur) et cliquer sur "OK".<br/>3. **Utiliser un modèle** : Cliquer sur "Fichier" puis "Nouveau", choisir un modèle (rapport, lettre, etc.) et cliquer sur "Créer".<br/>4. **Personnaliser un document** : Remplacer le texte du modèle par son propre contenu tout en conservant la mise en forme. |
| **Illustration ou schéma explicatif**         | **Schéma des options de styles dans Word** : <br/>• L'onglet **Accueil** contient la section "Styles". <br/>• La **galerie de styles** montre les différents styles disponibles avec un aperçu visuel. <br/>• Le menu **Fichier > Nouveau** permet d'accéder aux modèles.                                                                                   |
| **Exercice d'application ou question**        | **Exercice** : Demander à l'élève de créer un document sur les saisons de l'année : <br/>1. Appliquer le style "Titre 1" au titre principal <br/>2. Appliquer le style "Titre 2" aux noms des quatre saisons <br/>3. Modifier le style "Titre 2" pour changer la couleur selon chaque saison (vert pour printemps, jaune pour été, etc.) <br/>4. Créer une page de garde en utilisant un modèle                                                                                                                    |
| **Mot-clé ou astuce mnémotechnique**          | **STYLE** : <br/>**S**électionner le texte – **T**rouver le style dans la galerie – **Y**eux sur l'aperçu – **L**iquer pour appliquer – **E**nsemble cohérent dans tout le document                                                                                                                                                                                                                                                                                                                           |
