# FT-SI07 - Environnement matériel d’un système informatique

Cette fiche technique présente les différents environnements matériels dans lesquels un ordinateur peut fonctionner.

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                      |
|-----------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Environnement matériel d’un système informatique                                                                                                                                                                                |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                              |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Distinguer un ordinateur seul (monoposte) d’un réseau <br/>• Comprendre les composants matériels nécessaires                                                                     |
| **Définitions clés**                          | **Monoposte** : Ordinateur utilisé seul, non connecté à d’autres. <br/>**Réseau local (LAN)** : Plusieurs ordinateurs connectés entre eux. <br/>**Serveur** : Ordinateur central dans un réseau.                                  |
| **Explication de la notion avec exemples**    | À la maison, un ordinateur personnel est souvent monoposte. Au collège, les ordinateurs sont connectés en réseau local pour partager des ressources.                                                                            |
| **Méthodologie ou étapes à suivre**           | 1. Observer un ordinateur seul <br/>2. Observer un réseau d’ordinateurs <br/>3. Comparer les deux environnements                                                                                                                 |
| **Illustration ou schéma explicatif**         | Schéma comparant un ordinateur isolé et un réseau local avec plusieurs ordinateurs reliés.                                                                                                                                     |
| **Exercice d’application ou question**        | Demander à l’élève de dire si un exemple donné est un monoposte ou un réseau.                                                                                                                                                   |
| **Mot-clé ou astuce mnémotechnique**          | **MRL** : <br/>**M**onoposte <br/>**R**éseau <br/>**L**ocal                                                                                                                                                                     |
