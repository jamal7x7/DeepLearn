# FT-L1-03-inserer-images
Voici une fiche technique sous forme de tableau, conçue pour des élèves de 7e année (niveau 1), qui aborde la notion d'insertion et de manipulation d'images dans Microsoft Word 2021 :

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Insérer des Images dans Microsoft Word 2021                                                                                                                                                                                                                                                                                                                                                                                         |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Insérer une image depuis son appareil <br/>• Redimensionner une image <br/>• Déplacer une image dans le document <br/>• Appliquer des options de mise en page autour de l'image                                                                                                                                                                                                                |
| **Définitions clés**                          | **Insertion d'image** : Processus d'ajout d'une image dans un document. <br/>**Habillage du texte** : Façon dont le texte s'organise autour d'une image. <br/>**Redimensionnement** : Modification de la taille d'une image tout en conservant ses proportions.                                                                                                                                                                                          |
| **Explication de la notion avec exemples**    | L'élève insère une image dans son document pour illustrer son propos. Par exemple, dans un texte sur les animaux, il peut ajouter une photo de son animal préféré, la redimensionner pour qu'elle s'intègre harmonieusement dans la page, et faire en sorte que le texte s'organise autour de l'image.                                                                                                                       |
| **Méthodologie ou étapes à suivre**           | 1. **Insérer une image** : Cliquer à l'endroit souhaité, aller dans l'onglet "Insertion", cliquer sur "Images", puis "Ce appareil", sélectionner l'image et cliquer sur "Insérer".<br/>2. **Redimensionner** : Cliquer sur l'image et tirer sur les petits carrés aux coins pour agrandir ou réduire.<br/>3. **Déplacer** : Cliquer sur l'image et la faire glisser vers la nouvelle position.<br/>4. **Options de mise en page** : Cliquer droit sur l'image, choisir "Habillage du texte" et sélectionner "Carré" pour que le texte entoure l'image. |
| **Illustration ou schéma explicatif**         | **Schéma des options d'insertion d'image** : <br/>• L'onglet **Insertion** contient le bouton "Images". <br/>• Les **poignées de redimensionnement** apparaissent aux coins de l'image sélectionnée. <br/>• Le menu **Habillage du texte** montre différentes options d'organisation du texte autour de l'image.                                                                                   |
| **Exercice d'application ou question**        | **Exercice** : Demander à l'élève de créer un document sur son passe-temps préféré. Il doit : <br/>1. Insérer une image illustrant ce passe-temps <br/>2. Redimensionner l'image pour qu'elle occupe environ un quart de la page <br/>3. Positionner l'image à droite du texte <br/>4. Appliquer l'option d'habillage "Carré" pour que le texte entoure l'image                                                                                                                    |
| **Mot-clé ou astuce mnémotechnique**          | **IMAGE** : <br/>**I**nsertion – **M**ise en place – **A**justement de taille – **G**lissement (déplacement) – **E**nveloppement (habillage du texte)                                                                                                                                                                                                                                                                                                                           |
