# FT-L01-introduction-logo
Voici une fiche technique sous forme de tableau, conçue pour des élèves débutants, qui aborde les notions essentielles de la programmation avec le langage Logo :

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Découvrir le langage de programmation Logo                                                                                                                                                                                                                                                                                                                                                                                         |
| **Niveau concerné**                           | Débutant                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Comprendre ce qu'est le langage Logo <br/>• Connaître l'histoire et l'objectif de Logo <br/>• Identifier les éléments de base de l'interface Logo <br/>• Exécuter ses premières commandes de déplacement de la tortue                                                                                                                                                                                                                |
| **Définitions clés**                          | **Logo** : Langage de programmation éducatif créé dans les années 1960 pour initier les enfants à la programmation. <br/>**Tortue graphique** : Curseur en forme de triangle qui se déplace à l'écran selon les commandes données. <br/>**Commande** : Instruction donnée à la tortue pour qu'elle effectue une action spécifique.                                                                                                                                                                                          |
| **Explication de la notion avec exemples**    | Logo est un langage de programmation qui permet d'apprendre les bases de la programmation de façon visuelle. L'utilisateur donne des instructions à une "tortue" virtuelle qui dessine à l'écran. Par exemple, la commande `AV 100` fait avancer la tortue de 100 pas en laissant une trace, tandis que `TD 90` la fait tourner de 90 degrés vers la droite. Ces commandes simples permettent de créer des dessins géométriques et d'apprendre les concepts fondamentaux de la programmation.                                                                                                                       |
| **Méthodologie ou étapes à suivre**           | 1. **Lancer l'environnement Logo** : Ouvrir l'application Logo sur l'ordinateur.<br/>2. **Observer l'interface** : Identifier la zone de dessin (où se déplace la tortue) et la zone de commande (où l'on tape les instructions).<br/>3. **Premiers pas** : Taper des commandes simples comme `AV 50` (avancer de 50 pas) ou `TD 90` (tourner à droite de 90 degrés).<br/>4. **Effacer l'écran** : Utiliser la commande `VE` (vide écran) pour recommencer.<br/>5. **Expérimenter** : Combiner plusieurs commandes pour créer des formes simples. |
| **Illustration ou schéma explicatif**         | **Schéma des commandes de base de Logo** : <br/>• `AV n` : Avancer de n pas <br/>• `RE n` : Reculer de n pas <br/>• `TD n` : Tourner à droite de n degrés <br/>• `TG n` : Tourner à gauche de n degrés <br/>• `VE` : Vider l'écran (effacer tout) <br/>• `LC` : Lever le crayon (se déplacer sans dessiner) <br/>• `BC` : Baisser le crayon (dessiner en se déplaçant)                                                                                   |
| **Exercice d'application ou question**        | **Exercice** : Demander à l'élève de créer un carré en utilisant les commandes de base Logo. Solution : `AV 100 TD 90 AV 100 TD 90 AV 100 TD 90 AV 100 TD 90` ou plus simplement avec la commande de répétition : `REPETE 4 [AV 100 TD 90]`.                                                                                                                    |
| **Mot-clé ou astuce mnémotechnique**          | **LOGO** : <br/>**L**angage **O**rienté **G**raphique **O**rdonné - Un langage qui permet de donner des ordres graphiques de façon structurée.                                                                                                                                                                                                                                                                                                                           |
