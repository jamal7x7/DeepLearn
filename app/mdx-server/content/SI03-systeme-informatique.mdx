# Le Système Informatique

<ExerciseCard title="Découvrir les systèmes informatiques">

Dans cette leçon, tu vas découvrir ce qu'est un système informatique, comment il est organisé et comment ses différents composants interagissent pour traiter l'information.

Voici ce que tu vas apprendre :
- La définition et les caractéristiques d'un système informatique
- Les différents composants qui constituent un système informatique
- Les interactions entre ces composants
- Les différents types de systèmes informatiques et leurs applications

</ExerciseCard>

        <p>L'<strong>architecture</strong> d'un système informatique définit comment ses composants sont organisés et comment ils interagissent. On distingue généralement plusieurs couches :</p>
        <pre>
          <code>
{`+----------------------------------+
|           UTILISATEURS           |
+----------------------------------+
                 ↕
+----------------------------------+
|          APPLICATIONS            |
| (Logiciels, Services, Interfaces)|
+----------------------------------+
                 ↕
+----------------------------------+
|      SYSTÈME D'EXPLOITATION      |
|  (Gestion des ressources, APIs)  |
+----------------------------------+
                 ↕
+----------------------------------+
|            MATÉRIEL              |
| (CPU, Mémoire, Stockage, Réseau) |
+----------------------------------+`}
          </code>
        </pre>
        <p>Chaque couche fournit des services à la couche supérieure et utilise les services de la couche inférieure :</p>
        <ul>
          <li>Le <strong>matériel</strong> exécute les instructions de base</li>
          <li>Le <strong>système d'exploitation</strong> gère les ressources matérielles et fournit des services aux applications</li>
          <li>Les <strong>applications</strong> offrent des fonctionnalités aux utilisateurs</li>
          <li>Les <strong>utilisateurs</strong> interagissent avec le système via les interfaces des applications</li>
        </ul>
        <p>Les <strong>interfaces</strong> entre ces couches sont essentielles : elles définissent comment les composants communiquent entre eux. Par exemple, l'API (Interface de Programmation d'Application) d'un système d'exploitation permet aux applications d'accéder aux ressources matérielles sans avoir à connaître leurs détails techniques.</p>

## Exercice 1: Identification des composants

Pour chaque élément ci-dessous, indique à quelle catégorie de composant d'un système informatique il appartient (matériel, logiciel, données, réseau ou utilisateur) :
1. Un fichier PDF
2. Un routeur Wi-Fi
3. Un administrateur système
4. Microsoft Excel
5. Un câble Ethernet
6. Une base de données client

<details>
  <summary>Solution</summary>
  <p>1. Données</p>
  <p>2. Matériel (et réseau)</p>
  <p>3. Utilisateur</p>
  <p>4. Logiciel</p>
  <p>5. Réseau (et matériel)</p>
  <p>6. Données</p>
</details>

## Exercice 2: Architecture système

Dessine ou décris l'architecture d'un système informatique simple pour une petite bibliothèque. Le système doit permettre de gérer les livres, les emprunts et les membres de la bibliothèque.

<details>
  <summary>Solution</summary>
  <p>Architecture possible pour un système de gestion de bibliothèque :</p>
  <pre>
    <code>
        Matériel :
        - Serveur central pour héberger la base de données
        - Postes de travail pour le personnel
        - Scanners de codes-barres pour les livres
        - Imprimantes pour les reçus et les cartes de membre

        Logiciels :
        - Système d'exploitation sur tous les appareils
        - Logiciel de gestion de bibliothèque
        - Système de gestion de base de données
        - Logiciels utilitaires (sauvegarde, sécurité)

        Données :
        - Base de données des livres (titre, auteur, ISBN, etc.)
        - Base de données des membres (nom, coordonnées, etc.)
        - Historique des emprunts et retours

        Réseau :
        - Réseau local (LAN) reliant tous les postes
        - Connexion Internet pour l'accès au catalogue en ligne

        Utilisateurs :
        - Bibliothécaires (gestion des livres et des emprunts)
        - Administrateur système (maintenance)
        - Membres de la bibliothèque (consultation du catalogue)
    </code>
  </pre>
</details>

## Exercice 3: Analyse de système

Identifie un système informatique que tu utilises régulièrement (smartphone, console de jeux, ordinateur personnel, etc.) et analyse-le en identifiant :
1. Ses composants matériels principaux
2. Les logiciels qu'il utilise
3. Les types de données qu'il traite
4. Ses connexions réseau
5. Ses utilisateurs typiques

<details>
  <summary>Exemple de solution pour un smartphone</summary>
  <p>1. Composants matériels : processeur, mémoire RAM, stockage flash, écran tactile, caméras, haut-parleurs, microphones, batterie, antennes (4G/5G, Wi-Fi, Bluetooth, GPS)</p>
  <p>2. Logiciels : système d'exploitation (Android/iOS), applications préinstallées, applications téléchargées, pilotes</p>
  <p>3. Données : contacts, messages, photos, vidéos, documents, paramètres, historique de navigation</p>
  <p>4. Connexions réseau : réseau mobile (3G/4G/5G), Wi-Fi, Bluetooth, NFC</p>
  <p>5. Utilisateurs : propriétaire principal, éventuellement famille ou amis, applications tierces (via API)</p>
</details>

## Défis créatifs

1. **Conception d'un système** : Imagine et conçois un système informatique pour une petite entreprise de ton choix (restaurant, boutique, cabinet médical, etc.). Décris ses composants, son architecture et son fonctionnement.

2. **Analyse comparative** : Compare deux types de systèmes informatiques différents (par exemple, un système embarqué comme une montre connectée et un système d'entreprise). Identifie leurs similitudes et leurs différences en termes de composants, d'architecture et de fonctionnalités.

3. **Évolution historique** : Recherche l'évolution d'un type de système informatique (ordinateur personnel, serveur, smartphone, etc.) au cours des 20 dernières années. Quels ont été les changements majeurs dans ses composants et son architecture ?

## Réflexion

Les systèmes informatiques sont omniprésents dans notre société moderne. Réfléchis aux questions suivantes :

- Comment les systèmes informatiques ont-ils changé notre façon de vivre et de travailler ?
- Quels sont les avantages et les inconvénients de la dépendance croissante aux systèmes informatiques ?
- Comment les systèmes informatiques vont-ils évoluer dans le futur ? Quelles nouvelles technologies pourraient transformer leur architecture et leur fonctionnement ?
- Quels défis éthiques et sociétaux posent les systèmes informatiques modernes (vie privée, sécurité, accessibilité, etc.) ?
