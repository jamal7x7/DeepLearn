# FT-L09-validation-donnees
Voici une fiche technique sous forme de tableau, conçue pour des élèves de 7e année (niveau 1), qui aborde la notion de validation des données dans Microsoft Excel 2021 :

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Validation des Données dans Excel 2021                                                                                                                                                                                                                                                                                                                                                                                                         |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Comprendre l'importance de la validation des données <br/>• Créer des règles de validation simples (nombres, dates, listes) <br/>• Ajouter des messages d'entrée et d'erreur <br/>• Identifier et corriger les données non valides                                                                                                                                                                                                                |
| **Définitions clés**                          | **Validation des données** : Processus qui contrôle le type et la valeur des données saisies dans une cellule. <br/>**Critère de validation** : Règle définissant quelles données sont acceptables (plage de valeurs, format, etc.). <br/>**Message d'entrée** : Texte informatif qui s'affiche lorsqu'une cellule est sélectionnée. <br/>**Message d'erreur** : Alerte qui apparaît lorsqu'une donnée non valide est saisie. <br/>**Liste déroulante** : Menu permettant de choisir parmi des options prédéfinies.                                                                                                                                                                          |
| **Explication de la notion avec exemples**    | L'élève crée un formulaire de saisie pour un club sportif. Il utilise la validation des données pour s'assurer que les informations saisies sont correctes : l'âge doit être un nombre entre 8 et 18 ans, le sport pratiqué doit être choisi dans une liste prédéfinie (football, basketball, natation, etc.), et la date d'inscription doit être postérieure au 1er septembre 2023. Grâce à la validation, l'élève évite les erreurs de saisie et garantit la cohérence des données.                                                                                                                       |
| **Méthodologie ou étapes à suivre**           | 1. **Sélectionner les cellules** : Cliquer et glisser pour sélectionner les cellules auxquelles appliquer la validation.<br/>2. **Ouvrir la validation des données** : Dans l'onglet "Données", cliquer sur "Validation des données".<br/>3. **Définir les critères** : Dans l'onglet "Paramètres" : <br/>   - Choisir le type de données autorisées (nombre entier, décimal, liste, date, etc.) <br/>   - Définir les contraintes (entre, supérieur à, égal à, etc.) <br/>   - Spécifier les valeurs limites<br/>4. **Ajouter un message d'entrée** : Dans l'onglet "Message d'entrée" : <br/>   - Saisir un titre <br/>   - Rédiger un message explicatif qui s'affichera lorsque la cellule sera sélectionnée<br/>5. **Définir un message d'erreur** : Dans l'onglet "Alerte d'erreur" : <br/>   - Choisir le style (Arrêt, Avertissement, Information) <br/>   - Saisir un titre <br/>   - Rédiger un message qui s'affichera en cas de donnée non valide<br/>6. **Créer une liste déroulante** : <br/>   - Dans le type de données, sélectionner "Liste" <br/>   - Saisir les options séparées par des points-virgules ou référencer une plage de cellules contenant les options |
| **Illustration ou schéma explicatif**         | **Schéma de la validation des données** : <br/>• La boîte de dialogue **Validation des données** comporte trois onglets : Paramètres, Message d'entrée et Alerte d'erreur. <br/>• L'onglet **Paramètres** permet de définir le type de données et les contraintes. <br/>• L'onglet **Message d'entrée** configure l'aide contextuelle. <br/>• L'onglet **Alerte d'erreur** définit le message en cas d'erreur. <br/>• Le bouton **Cercle les données non valides** dans le menu Validation permet d'identifier visuellement les erreurs.                                                                                   |
| **Exercice d'application ou question**        | **Exercice** : Demander à l'élève de créer un formulaire d'inscription à une sortie scolaire contenant : <br/>1. Nom et prénom (texte libre) <br/>2. Classe (liste déroulante : 6e, 7e, 8e) <br/>3. Âge (nombre entier entre 11 et 15) <br/>4. Date de naissance (date, antérieure au 01/01/2013) <br/>5. Activité choisie (liste déroulante : Musée, Cinéma, Parc d'attractions) <br/>6. Autorisation parentale (liste déroulante : Oui, Non) <br/>Pour chaque champ avec validation : <br/>• Ajouter un message d'entrée expliquant ce qui doit être saisi <br/>• Définir un message d'erreur approprié <br/>• Tester le formulaire en saisissant des données valides et non valides                                                                                                                    |
| **Mot-clé ou astuce mnémotechnique**          | **VALIDE** : <br/>**V**érifier les données – **A**jouter des contraintes – **L**iste déroulante pour les choix – **I**nformer avec des messages – **D**étecter les erreurs – **E**mpêcher les saisies incorrectes                                                                                                                                                                                                                                                                                                                           |
