# IV- Les procédures :
---

### 1- Définition :

Une procédure « xlogo » est un petit programme qui commence par
le mot « pour » et se fini par le mot « Fin ».

Une procédure peut s’écrire sous la forme :
```logo
POUR nom_de_la_procedure
Primitives à exécuter
FIN
```
### 2- Exemple : 

La procédure pour dessiner un carré est :
```logo
POUR carre
Repete 4 [ AV 100 TD 90]
FIN
```

Pour que la tortue dessine le carré il suffit d’écrire le nom de la procédure :
carre et faire entrée


### 3- Remarque : xlogo se rappelle de toutes les procédures crées , pour supprimer
une prcédure on écrit : `effacenom" nom`

### 4- Exemple : pour effacer la procédure « carre » on écrit :
```logo
   Effacenom" carre
```
