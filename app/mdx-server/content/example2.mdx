# Example 2: Code Blocks and Tables

## Code Blocks

Here's a JavaScript code block:

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet('World');
```

And a Python code block:

```python
def add(a, b):
  return a + b

print(add(5, 3))
```
