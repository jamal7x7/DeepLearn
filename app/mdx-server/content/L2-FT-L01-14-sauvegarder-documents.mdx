# FT-L01-14-sauvegarder-documents

Cette fiche technique explique comment sauvegarder un document dans Microsoft Word 2021 pour éviter toute perte de données.

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                      |
|-----------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Sauvegarder un document Word                                                                                                                                                                                                    |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                              |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Enregistrer un document <br/>• Choisir un emplacement de sauvegarde <br/>• Comprendre la différence entre "Enregistrer" et "Enregistrer sous"                                   |
| **Définitions clés**                          | **Sauvegarder** : Enregistrer un document pour ne pas perdre son travail. <br/>**Enregistrer sous** : Enregistrer une copie du document sous un autre nom ou dans un autre emplacement.                                          |
| **Explication de la notion avec exemples**    | Après avoir tapé un texte, l’élève clique sur l’icône de disquette pour enregistrer. Pour créer une copie, il utilise "Enregistrer sous" et choisit un nouveau nom ou un autre dossier.                                         |
| **Méthodologie ou étapes à suivre**           | 1. Cliquer sur **Fichier** <br/>2. Sélectionner **Enregistrer** ou **Enregistrer sous** <br/>3. Choisir l’emplacement (ordinateur, clé USB, cloud) <br/>4. Donner un nom au fichier <br/>5. Cliquer sur **Enregistrer**         |
| **Illustration ou schéma explicatif**         | **Icône disquette** dans la barre d’outils rapide. Fenêtre **Enregistrer sous** avec choix du dossier et nom du fichier.                                                                                                       |
| **Exercice d’application ou question**        | Demander à l’élève de créer un document, de l’enregistrer sous le nom "MonPremierTexte", puis d’en faire une copie nommée "CopieMonPremierTexte" dans un autre dossier.                                                          |
| **Mot-clé ou astuce mnémotechnique**          | **SAVE** : <br/>**S**auvegarder <br/>**A**ppuyer sur disquette <br/>**V**érifier l’emplacement <br/>**E**nregistrer sous un nom clair                                                                                           |
