# FT-U03-recherche-fichier-support
Voici une fiche technique sous forme de tableau, conçue pour des élèves de 7e année (niveau 1), qui aborde la notion de recherche d'un fichier sur un support de stockage :

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Recherche d'un fichier sur un support de stockage                                                                                                                                                                                                                                                                                                                                                                         |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Identifier les différents supports de stockage <br/>• Naviguer dans l'arborescence des dossiers <br/>• Utiliser les outils de recherche du système d'exploitation <br/>• Retrouver un fichier selon différents critères (nom, date, type)                                                                                                                                                                                                                |
| **Définitions clés**                          | **Support de stockage** : Dispositif physique ou virtuel permettant de conserver des données numériques (disque dur, clé USB, carte SD, stockage cloud). <br/>**Arborescence** : Organisation hiérarchique des dossiers et fichiers, représentée sous forme d'un arbre avec des branches. <br/>**Explorateur de fichiers** : Programme permettant de visualiser, organiser et rechercher des fichiers et dossiers sur un ordinateur.                                                                                                                                                                          |
| **Explication de la notion avec exemples**    | L'élève découvre que les fichiers numériques sont organisés dans une structure hiérarchique sur différents supports de stockage. Par exemple, sur un ordinateur Windows, l'explorateur de fichiers permet de naviguer entre le disque local (C:), les périphériques externes (clés USB, disques externes) et les dossiers personnels (Documents, Images, etc.). Pour retrouver un fichier, l'élève peut parcourir manuellement cette arborescence ou utiliser la fonction de recherche intégrée qui permet de filtrer les résultats selon le nom, l'extension, la date de modification ou la taille du fichier.                                                                                                                       |
| **Méthodologie ou étapes à suivre**           | 1. **Identifier le support de stockage** : Déterminer si le fichier se trouve sur le disque dur interne, une clé USB, un disque externe ou dans le cloud.<br/>2. **Ouvrir l'explorateur de fichiers** : Sur Windows, utiliser l'icône du dossier dans la barre des tâches ou appuyer sur Windows+E. Sur macOS, cliquer sur l'icône du Finder.<br/>3. **Naviguer dans l'arborescence** : Cliquer sur les dossiers pour les ouvrir et utiliser les boutons de navigation (précédent, suivant, niveau supérieur).<br/>4. **Utiliser la recherche** : Cliquer dans la barre de recherche en haut à droite et saisir tout ou partie du nom du fichier.<br/>5. **Affiner la recherche** : Utiliser les filtres avancés pour préciser le type de fichier, la date de modification ou la taille. |
| **Illustration ou schéma explicatif**         | **Schéma de l'explorateur de fichiers** : <br/>• À gauche, le **volet de navigation** montrant les différents supports et dossiers principaux. <br/>• Au centre, la **liste des fichiers et dossiers** du répertoire sélectionné. <br/>• En haut, la **barre d'adresse** indiquant l'emplacement actuel dans l'arborescence. <br/>• En haut à droite, la **barre de recherche** permettant de trouver rapidement des fichiers.                                                                                   |
| **Exercice d'application ou question**        | **Exercice** : Demander à l'élève de retrouver un fichier spécifique en suivant ces étapes : <br/>1. Ouvrir l'explorateur de fichiers <br/>2. Naviguer jusqu'au dossier Documents <br/>3. Utiliser la barre de recherche pour trouver tous les documents PDF créés le mois dernier <br/>4. Trier les résultats par date de modification <br/>5. Localiser un fichier spécifique et noter son chemin complet (par exemple : C:\Users\[nom]\Documents\Cours\fichier.pdf) <br/>6. Trouver un moyen alternatif d'accéder directement à ce fichier                                                                                                                    |
| **Mot-clé ou astuce mnémotechnique**          | **SAFER** : <br/>**S**upport de stockage – **A**rborescence à parcourir – **F**iltres de recherche – **E**xplorateur de fichiers – **R**echerche par critères                                                                                                                                                                                                                                                                                                                           |
