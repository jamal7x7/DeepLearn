# FT-L05-creation-graphiques
Voici une fiche technique sous forme de tableau, conçue pour des élèves de 7e année (niveau 1), qui aborde la notion de création et personnalisation des graphiques dans Microsoft Excel 2021 :

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Création et Personnalisation des Graphiques dans Excel 2021                                                                                                                                                                                                                                                                                                                                                                                                         |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Créer différents types de graphiques (histogrammes, secteurs, courbes) <br/>• Sélectionner les données appropriées pour un graphique <br/>• Personnaliser l'apparence d'un graphique (titres, légendes, couleurs) <br/>• Interpréter les informations présentées dans un graphique                                                                                                                                                                                                                |
| **Définitions clés**                          | **Graphique** : Représentation visuelle de données numériques. <br/>**Histogramme** : Graphique à barres verticales comparant des valeurs entre différentes catégories. <br/>**Graphique en secteurs** : Représentation circulaire (camembert) montrant la proportion de chaque valeur par rapport au total. <br/>**Graphique en courbes** : Représentation de l'évolution de valeurs sur une période de temps. <br/>**Légende** : Explication des éléments représentés dans le graphique.                                                                                                                                                                          |
| **Explication de la notion avec exemples**    | L'élève crée un tableau de données sur les ventes mensuelles de différents produits, puis représente ces données sous forme de graphique pour mieux visualiser les tendances. Par exemple, un histogramme permet de comparer les ventes de chaque produit mois par mois, un graphique en secteurs montre la répartition des ventes totales par produit, et un graphique en courbes illustre l'évolution des ventes d'un produit spécifique au fil du temps. Ces représentations visuelles facilitent l'analyse et l'interprétation des données.                                                                                                                       |
| **Méthodologie ou étapes à suivre**           | 1. **Préparer les données** : Organiser les données dans un tableau avec des en-têtes clairs pour les lignes et les colonnes.<br/>2. **Sélectionner les données** : Cliquer et glisser pour sélectionner toutes les données à inclure dans le graphique, y compris les en-têtes.<br/>3. **Insérer un graphique** : Dans l'onglet "Insertion", cliquer sur le type de graphique souhaité dans le groupe "Graphiques".<br/>4. **Choisir une disposition** : Sélectionner parmi les dispositions proposées celle qui convient le mieux aux données.<br/>5. **Personnaliser le graphique** : <br/>   - Ajouter un titre au graphique (onglet "Création de graphique") <br/>   - Modifier les étiquettes des axes (onglet "Mise en forme") <br/>   - Ajuster les couleurs et le style (onglet "Création de graphique") <br/>   - Ajouter ou supprimer la légende (onglet "Mise en forme")<br/>6. **Analyser le graphique** : Observer les tendances, les comparaisons ou les proportions mises en évidence par le graphique. |
| **Illustration ou schéma explicatif**         | **Schéma des éléments d'un graphique** : <br/>• Le **titre du graphique** en haut indique le sujet représenté. <br/>• Les **axes** horizontal (X) et vertical (Y) définissent les dimensions du graphique. <br/>• Les **étiquettes d'axes** précisent ce que représente chaque axe. <br/>• La **légende** explique la signification des couleurs ou motifs utilisés. <br/>• Les **séries de données** sont les ensembles de valeurs représentés (barres, secteurs, lignes). <br/>• La **zone de traçage** est l'espace où les données sont représentées graphiquement.                                                                                   |
| **Exercice d'application ou question**        | **Exercice** : Demander à l'élève de créer un tableau des températures moyennes mensuelles de sa ville sur une année, puis de : <br/>1. Créer un graphique en courbes montrant l'évolution des températures au fil des mois <br/>2. Ajouter un titre approprié au graphique <br/>3. Étiqueter correctement les axes (X pour les mois, Y pour les températures en °C) <br/>4. Personnaliser les couleurs de la courbe <br/>5. Ajouter des étiquettes de données pour afficher les valeurs exactes <br/>6. Créer un deuxième graphique (histogramme) avec les mêmes données <br/>7. Comparer les deux représentations et expliquer laquelle est la plus adaptée pour montrer l'évolution des températures                                                                                                                    |
| **Mot-clé ou astuce mnémotechnique**          | **GRAPHE** : <br/>**G**énérer le visuel – **R**eprésenter les données – **A**jouter des titres et légendes – **P**ersonnaliser l'apparence – **H**armoniser les couleurs – **E**xpliquer les tendances                                                                                                                                                                                                                                                                                                                           |
