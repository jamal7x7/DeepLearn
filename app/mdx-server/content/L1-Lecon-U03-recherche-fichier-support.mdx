# Recherche d'un fichier sur un support de stockage

<ExerciseCard title="Découvrir les supports de stockage et l'organisation des fichiers">

## Introduction

Bienvenue dans cette leçon sur la recherche de fichiers ! Aujourd'hui, nous allons apprendre à retrouver efficacement des documents numériques sur différents supports de stockage. Que tu cherches un devoir, une photo ou un document important, ces compétences te seront utiles au quotidien.

## Qu'est-ce qu'un support de stockage ?

Un support de stockage est un dispositif physique ou virtuel qui permet de conserver des données numériques. Les fichiers que tu crées sont enregistrés sur ces supports. Voici les principaux types :

- **Disque dur interne** : Intégré à ton ordinateur, c'est là que sont stockés ton système d'exploitation et tes programmes
- **Clé USB** : Petit dispositif portable qui se branche sur un port USB
- **Carte SD** : Utilisée principalement dans les appareils photo, smartphones et tablettes
- **Disque dur externe** : Plus grande capacité qu'une clé USB, idéal pour les sauvegardes
- **Stockage cloud** : Espace de stockage en ligne (Google Drive, OneDrive, Dropbox)

## L'organisation des fichiers en arborescence

Les fichiers sont organisés de manière hiérarchique, comme un arbre avec ses branches :

- Le **disque** (C:, D:, etc.) représente la racine
- Les **dossiers** (ou répertoires) sont les branches principales
- Les **sous-dossiers** sont les branches secondaires
- Les **fichiers** sont comme les feuilles de l'arbre

Cette organisation s'appelle une **arborescence**. Par exemple, un fichier peut se trouver à l'emplacement :
`C:\Utilisateurs\Ton_nom\Documents\Ecole\Informatique\devoir.docx`

Ce chemin indique que le fichier "devoir.docx" se trouve dans le sous-dossier "Informatique", lui-même dans "Ecole", dans "Documents", etc.

## L'explorateur de fichiers : ton outil de navigation

L'explorateur de fichiers (Windows) ou le Finder (macOS) est le programme qui te permet de visualiser et d'organiser tes fichiers.

## Comment l'ouvrir ?

- **Sur Windows** : Clique sur l'icône du dossier dans la barre des tâches ou appuie sur la touche Windows + E
- **Sur macOS** : Clique sur l'icône du Finder dans le Dock (le visage souriant bleu)

## Les éléments de l'interface

Familiarise-toi avec ces éléments essentiels :

1. **Volet de navigation** (à gauche) : Affiche les emplacements principaux (Ce PC, Documents, Images...)
2. **Barre d'adresse** (en haut) : Montre ton emplacement actuel dans l'arborescence
3. **Zone principale** (au centre) : Affiche le contenu du dossier sélectionné
4. **Barre de recherche** (en haut à droite) : Permet de rechercher des fichiers
5. **Boutons de navigation** : Permettent d'aller en arrière, en avant ou de remonter d'un niveau

## Exercice pratique

1. Ouvre l'explorateur de fichiers sur ton ordinateur
2. Navigue jusqu'au dossier "Documents"
3. Crée un nouveau dossier nommé "Exercices_Recherche"
4. Ouvre ce dossier
5. Remonte d'un niveau (retour à Documents)
6. Utilise le volet de navigation pour aller directement dans "Images"
7. Reviens en arrière (bouton précédent)

Cette navigation te permettra de te familiariser avec l'interface et les déplacements dans l'arborescence.

<ExerciseCard title="Rechercher un fichier par son nom">

## La recherche par nom : la méthode la plus directe

Lorsque tu connais le nom (ou une partie du nom) du fichier que tu cherches, la recherche par nom est la méthode la plus efficace.

## Étape 1 : Choisir le point de départ

1. Ouvre l'explorateur de fichiers
2. Navigue jusqu'à l'emplacement où tu penses que le fichier pourrait se trouver
   - Si tu n'es pas sûr, commence par un emplacement plus général comme "Ce PC" (Windows) ou le disque dur (macOS)

## Étape 2 : Utiliser la barre de recherche

1. Clique dans la barre de recherche en haut à droite
2. Tape tout ou partie du nom du fichier
   - Tu n'as pas besoin de connaître le nom exact
   - La recherche trouvera tous les fichiers contenant les caractères que tu as saisis

## Étape 3 : Attendre les résultats

1. La recherche s'effectue automatiquement pendant que tu tapes
2. Les résultats s'affichent dans la zone principale
3. Plus tu ajoutes de caractères, plus la recherche se précise

## Astuces pour la recherche par nom

- Utilise des **caractères génériques** comme l'astérisque (*) pour remplacer plusieurs caractères
  - Exemple : "devoir*.docx" trouvera "devoir1.docx", "devoirMaths.docx", etc.
- La recherche n'est généralement **pas sensible à la casse** (majuscules/minuscules)
- Si tu ne trouves pas ton fichier, essaie avec **moins de caractères** pour élargir la recherche

## Exercice pratique

1. Dans l'explorateur de fichiers, navigue jusqu'au dossier "Documents"
2. Dans la barre de recherche, tape "exercice"
3. Observe les résultats qui s'affichent
4. Affine ta recherche en ajoutant d'autres caractères (par exemple "exercice math")
5. Essaie d'utiliser un caractère générique (par exemple "ex*.pdf")

<ExerciseCard title="Rechercher un fichier selon d'autres critères">

## Au-delà du nom : les critères avancés

Parfois, tu ne te souviens pas du nom exact d'un fichier, mais tu connais d'autres informations comme sa date de création, son type ou sa taille.

## Recherche par type de fichier

1. Dans la barre de recherche, tape l'extension du fichier précédée d'un point
   - Exemple : ".pdf" pour tous les documents PDF
   - Exemple : ".jpg" ou ".png" pour les images

## Recherche par date

### Sur Windows :
1. Dans la barre de recherche, tape "date:" suivi d'une période
   - Exemple : "date:cette semaine" ou "date:mois dernier"
2. Ou utilise les filtres avancés en cliquant sur "Date de modification" dans le ruban

### Sur macOS :
1. Dans le Finder, clique sur le bouton + en haut à droite de la fenêtre
2. Sélectionne "Date de création" ou "Date de modification"
3. Choisis une période dans le menu déroulant

## Recherche par taille

### Sur Windows :
1. Dans la barre de recherche, tape "taille:" suivi d'une indication
   - Exemple : "taille:grand" pour les fichiers volumineux
   - Exemple : "taille:>10MB" pour les fichiers de plus de 10 Mo

### Sur macOS :
1. Dans le Finder, clique sur le bouton + en haut à droite
2. Sélectionne "Taille du fichier"
3. Choisis une option dans le menu déroulant

## Combinaison de critères

Tu peux combiner plusieurs critères pour affiner ta recherche :
- Exemple : ".docx date:cette semaine" pour trouver les documents Word créés cette semaine
- Exemple : "rapport taille:>5MB" pour trouver les gros fichiers contenant le mot "rapport"

## Exercice pratique

1. Trouve tous les fichiers PDF créés le mois dernier
2. Recherche les images de grande taille (plus de 5 Mo)
3. Trouve tous les documents Word contenant le mot "projet" créés cette année

<ExerciseCard title="Rechercher sur différents supports de stockage">

## Élargir ta recherche à d'autres supports

Les fichiers peuvent être stockés sur différents supports. Voici comment effectuer des recherches sur chacun d'eux.

## Sur le disque dur interne

1. Ouvre l'explorateur de fichiers
2. Sélectionne "Ce PC" (Windows) ou le nom de ton disque dur (macOS)
3. Lance ta recherche depuis cet emplacement pour chercher sur tout le disque

## Sur une clé USB ou un disque externe

1. Branche ta clé USB ou ton disque externe
2. Attends que le périphérique soit reconnu
3. Dans l'explorateur, sélectionne la lettre du lecteur (par exemple E: ou F:)
4. Utilise la barre de recherche pour chercher sur ce support spécifique

## Dans le stockage cloud

### Pour OneDrive, Google Drive ou Dropbox :
1. Ouvre l'application de synchronisation sur ton ordinateur
2. Utilise la fonction de recherche intégrée

### Ou directement dans le navigateur :
1. Connecte-toi à ton service de stockage cloud
2. Utilise la barre de recherche du site web

## Astuces pour les recherches multi-supports

- Vérifie que les supports externes sont bien connectés avant de lancer la recherche
- Les recherches sur des supports externes peuvent être plus lentes
- Pour le cloud, une connexion internet est nécessaire
- Certains services cloud ont des options de recherche avancées (par exemple, la recherche dans le contenu des documents)

## Exercice pratique

1. Branche une clé USB (si disponible)
2. Effectue une recherche pour trouver tous les documents texte (.txt, .docx) sur cette clé
3. Si tu utilises un service de stockage cloud, connecte-toi et recherche un fichier que tu as récemment modifié

<ExerciseCard title="Projet pratique : Retrouver des fichiers perdus">

## Objectif

Mets en pratique ce que tu as appris en relevant un défi de recherche de fichiers. Ce projet te permettra d'utiliser toutes les compétences acquises dans cette leçon.

## Scénario

Imagine que tu dois retrouver plusieurs fichiers importants dispersés sur ton ordinateur :

1. Un document PDF contenant le mot "rapport" créé il y a environ un mois
2. Une image dont le nom contient "vacances" et qui pèse plus de 2 Mo
3. Un fichier texte ou document modifié hier ou aujourd'hui
4. N'importe quel fichier créé par toi la semaine dernière

## Étapes du projet

1. **Planification** :
   - Pour chaque fichier à retrouver, note les critères de recherche que tu vas utiliser
   - Détermine le support de stockage le plus probable pour chaque fichier

2. **Recherche** :
   - Utilise les techniques appropriées pour chaque type de recherche
   - Note le chemin complet de chaque fichier trouvé

3. **Optimisation** :
   - Essaie de trouver une méthode plus rapide pour accéder à ces fichiers à l'avenir
   - Crée des raccourcis ou réorganise tes fichiers si nécessaire

## Critères de réussite

Ton projet est réussi si :
- Tu as retrouvé au moins 3 des 4 types de fichiers demandés
- Tu as noté le chemin complet de chaque fichier
- Tu as proposé au moins une méthode pour accéder plus rapidement à ces fichiers à l'avenir

## Pour aller plus loin

Si tu as terminé et que tu veux explorer davantage :
- Crée un système d'organisation de fichiers efficace avec des dossiers bien nommés
- Apprends à utiliser des outils de recherche plus avancés comme "Everything" (Windows) ou Spotlight (macOS)
- Explore les options d'indexation de fichiers pour accélérer les recherches futures
