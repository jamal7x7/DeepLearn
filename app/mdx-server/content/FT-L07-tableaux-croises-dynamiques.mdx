# FT-L07-tableaux-croises-dynamiques
Voici une fiche technique sous forme de tableau, conçue pour des élèves de 7e année (niveau 1), qui aborde la notion de tableaux croisés dynamiques dans Microsoft Excel 2021 :

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Tableaux Croisés Dynamiques dans Excel 2021                                                                                                                                                                                                                                                                                                                                                                                                         |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Comprendre le concept de tableau croisé dynamique <br/>• Créer un tableau croisé dynamique simple <br/>• Organiser les données en lignes, colonnes, valeurs et filtres <br/>• Modifier la présentation et les calculs d'un tableau croisé dynamique                                                                                                                                                                                                                |
| **Définitions clés**                          | **Tableau croisé dynamique** : Outil d'analyse permettant de résumer et d'explorer des données selon différentes perspectives. <br/>**Champ de ligne** : Catégorie affichée verticalement dans le tableau croisé. <br/>**Champ de colonne** : Catégorie affichée horizontalement dans le tableau croisé. <br/>**Champ de valeurs** : Données numériques à analyser (sommes, moyennes, etc.). <br/>**Champ de filtre** : Permet de filtrer l'ensemble du tableau croisé selon un critère.                                                                                                                                                                          |
| **Explication de la notion avec exemples**    | L'élève dispose d'un tableau contenant les ventes mensuelles de différents produits par région. Avec un tableau croisé dynamique, il peut rapidement synthétiser ces données pour répondre à des questions comme : "Quelles sont les ventes totales par région ?", "Comment les ventes de chaque produit évoluent-elles par mois ?", ou "Quelle région a vendu le plus d'un produit spécifique ?". Le tableau croisé permet de réorganiser les données sans modifier le tableau d'origine, offrant ainsi différentes perspectives d'analyse.                                                                                                                       |
| **Méthodologie ou étapes à suivre**           | 1. **Préparer les données** : S'assurer que le tableau source a des en-têtes clairs et ne contient pas de cellules vides.<br/>2. **Créer le tableau croisé** : <br/>   - Sélectionner n'importe quelle cellule du tableau source <br/>   - Dans l'onglet "Insertion", cliquer sur "Tableau croisé dynamique" <br/>   - Vérifier la plage de données sélectionnée <br/>   - Choisir l'emplacement du tableau croisé (nouvelle feuille ou feuille existante)<br/>3. **Construire le tableau croisé** : <br/>   - Dans le volet "Champs de tableau croisé dynamique", faire glisser les champs vers les zones appropriées : <br/>   - Lignes : catégories à afficher verticalement <br/>   - Colonnes : catégories à afficher horizontalement <br/>   - Valeurs : données à calculer (somme, moyenne, etc.) <br/>   - Filtres : champs pour filtrer l'ensemble du tableau<br/>4. **Modifier les calculs** : <br/>   - Cliquer droit sur une valeur dans le tableau <br/>   - Sélectionner "Paramètres des champs de valeurs" <br/>   - Choisir le type de calcul (Somme, Moyenne, Nombre, etc.)<br/>5. **Actualiser les données** : Si les données source changent, cliquer droit sur le tableau croisé et sélectionner "Actualiser" |
| **Illustration ou schéma explicatif**         | **Schéma d'un tableau croisé dynamique** : <br/>• Le **volet des champs** à droite permet de glisser-déposer les champs dans les différentes zones. <br/>• La zone **Filtres** en haut permet de filtrer l'ensemble du tableau. <br/>• La zone **Colonnes** définit les en-têtes horizontaux. <br/>• La zone **Lignes** définit les en-têtes verticaux. <br/>• La zone **Valeurs** contient les données calculées à l'intersection des lignes et colonnes. <br/>• Les **boutons +/-** permettent de développer ou réduire les niveaux de détail.                                                                                   |
| **Exercice d'application ou question**        | **Exercice** : Demander à l'élève de créer un tableau de données sur les résultats d'une enquête auprès des élèves de l'école contenant : <br/>1. Classe (6e, 7e, 8e) <br/>2. Genre (fille, garçon) <br/>3. Sport préféré (football, basketball, natation, athlétisme, autre) <br/>4. Temps hebdomadaire consacré au sport (en heures) <br/>Puis lui demander de : <br/>• Créer un tableau croisé dynamique montrant le nombre d'élèves par sport et par classe <br/>• Modifier le tableau pour afficher le temps moyen consacré au sport par genre et par sport <br/>• Ajouter un filtre pour pouvoir analyser les données par classe <br/>• Créer un second tableau croisé montrant la répartition des sports préférés selon le genre                                                                                                                    |
| **Mot-clé ou astuce mnémotechnique**          | **CROISE** : <br/>**C**atégoriser les données – **R**éorganiser sans modifier la source – **O**rganiser en lignes et colonnes – **I**nsérer des calculs – **S**ynthétiser l'information – **E**xplorer différentes perspectives                                                                                                                                                                                                                                                                                                                           |
