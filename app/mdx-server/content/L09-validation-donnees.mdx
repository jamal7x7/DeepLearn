# Validation des Données dans Excel 2021

<ExerciseCard title="Garantir la qualité de vos données">

Dans cette leçon, tu vas découvrir comment contrôler les données saisies dans tes feuilles de calcul pour éviter les erreurs et garantir la cohérence de tes informations.

Voici ce que tu vas apprendre :
- Comment définir des règles de validation pour différents types de données
- Comment créer des listes déroulantes pour limiter les choix possibles
- Comment ajouter des messages d'aide et d'erreur
- Comment identifier et corriger les données non valides

</ExerciseCard>

<Tip>
Tu peux combiner la validation des données avec la mise en forme conditionnelle pour mettre en évidence visuellement les cellules contenant des valeurs particulières, comme celles qui sont proches des limites définies dans tes règles de validation.
</Tip>

<ExerciseCard title="Exercice pratique: Formulaire d'inscription">

Crée un formulaire d'inscription à une sortie scolaire en suivant ces étapes :

1. Crée un tableau avec les en-têtes suivants :
   - Nom et prénom
   - Classe
   - Âge
   - Date de naissance
   - Activité choisie
   - Autorisation parentale

2. Applique les règles de validation suivantes :

   - Pour la colonne "Classe" :
     - Crée une liste déroulante avec les options : 6e, 7e, 8e
     - Ajoute un message d'entrée : "Sélectionne ta classe dans la liste"

   - Pour la colonne "Âge" :
     - Limite les valeurs aux nombres entiers entre 11 et 15
     - Ajoute un message d'erreur de style "Arrêt" : "L'âge doit être compris entre 11 et 15 ans"

   - Pour la colonne "Date de naissance" :
     - Vérifie que la date est antérieure au 01/01/2013
     - Ajoute un message d'entrée approprié

   - Pour la colonne "Activité choisie" :
     - Crée une liste déroulante avec : Musée, Cinéma, Parc d'attractions
     - Ajoute un message d'entrée explicatif

   - Pour la colonne "Autorisation parentale" :
     - Crée une liste déroulante avec : Oui, Non
     - Ajoute un message d'erreur indiquant que l'autorisation est obligatoire

3. Teste ton formulaire en saisissant des données valides et non valides

4. Utilise la fonction "Cercler les données non valides" pour vérifier ton travail

Bonus : Ajoute une mise en forme conditionnelle pour colorer en vert les cellules "Autorisation parentale" contenant "Oui" et en rouge celles contenant "Non".
