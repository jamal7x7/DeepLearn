# Fiche Technique : Réseaux Sans Fil

| **Titre du thème**                            | Réseaux Sans Fil                                                                                                                                                                                                                                                                                                                                                                                                         |
| **Niveau concerné**                           | Intermédiaire                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Identifier les différentes technologies de réseaux sans fil <br/>• Comprendre les principes de fonctionnement du Wi-Fi <br/>• Reconnaître les composants d'un réseau sans fil <br/>• Appliquer les bonnes pratiques de sécurité pour les réseaux sans fil                                                                                                                                                                                                                |
| **Définitions clés**                          | **Wi-Fi** : Technologie de réseau sans fil basée sur les normes IEEE 802.11. <br/>**SSID** : Nom qui identifie un réseau sans fil. <br/>**Point d'accès** : Équipement qui permet aux appareils de se connecter à un réseau sans fil. <br/>**Bande de fréquence** : Plage de fréquences radio utilisée pour la transmission (2,4 GHz, 5 GHz, 6 GHz). <br/>**WPA3** : Protocole de sécurité pour les réseaux Wi-Fi (Wi-Fi Protected Access 3).                                                                                                          |
| **Explication de la notion avec exemples**    | Les réseaux sans fil permettent aux appareils de communiquer sans connexion physique par câble. La technologie Wi-Fi est la plus répandue pour les réseaux locaux sans fil. Elle utilise des ondes radio pour transmettre les données entre les appareils et un point d'accès. Par exemple, lorsque tu utilises ton smartphone pour naviguer sur Internet à la maison, il se connecte probablement à un routeur Wi-Fi qui diffuse un signal sans fil. Ce signal est identifié par un nom (SSID) comme "Maison-Wifi" et protégé par un mot de passe. Les normes Wi-Fi ont évolué au fil du temps, offrant des débits de plus en plus élevés : Wi-Fi 4 (802.11n) jusqu'à 600 Mbps, Wi-Fi 5 (802.11ac) jusqu'à 3,5 Gbps, et Wi-Fi 6 (802.11ax) jusqu'à 9,6 Gbps. Le Wi-Fi utilise principalement deux bandes de fréquences : 2,4 GHz (plus grande portée mais plus d'interférences) et 5 GHz (moins d'interférences mais portée réduite). Le récent Wi-Fi 6E ajoute la bande 6 GHz pour encore moins d'interférences.                                                                                                                       |
| **Méthodologie ou étapes à suivre**           | 1. **Analyse des besoins** : Déterminer la zone à couvrir et le nombre d'appareils à connecter.<br/>2. **Choix de la norme Wi-Fi** : Sélectionner la norme adaptée aux besoins (Wi-Fi 5, Wi-Fi 6, etc.).<br/>3. **Planification du placement** : Positionner les points d'accès pour une couverture optimale.<br/>4. **Configuration du réseau** : Définir le SSID, le mot de passe et les paramètres de sécurité.<br/>5. **Test de couverture** : Vérifier la qualité du signal dans toutes les zones.<br/>6. **Optimisation** : Ajuster les canaux et la puissance pour éviter les interférences.<br/>7. **Sécurisation** : Mettre en place les protocoles de sécurité appropriés (WPA3). |

| **Conseils pratiques ou astuces**             | • Placez le routeur Wi-Fi dans un endroit central et en hauteur pour une meilleure couverture.<br/>• Évitez les obstacles comme les murs épais, les miroirs ou les grands objets métalliques.<br/>• Utilisez des canaux Wi-Fi différents de ceux de vos voisins pour réduire les interférences.<br/>• Pour les grandes surfaces, utilisez plusieurs points d'accès ou un système Wi-Fi maillé.<br/>• Activez la fonctionnalité "band steering" si disponible pour répartir automatiquement les appareils entre 2,4 GHz et 5 GHz.<br/>• Mettez régulièrement à jour le firmware de votre routeur pour bénéficier des dernières améliorations de sécurité. |
| **Erreurs courantes à éviter**                | • Utiliser un mot de passe Wi-Fi trop simple ou le laisser par défaut.<br/>• Négliger d'activer le chiffrement (utiliser WPA3 ou au minimum WPA2).<br/>• Placer le routeur près de sources d'interférences (micro-ondes, téléphones sans fil, etc.).<br/>• Diffuser le SSID avec des informations personnelles identifiables.<br/>• Oublier de désactiver l'accès administrateur à distance sur le routeur.<br/>• Surcharger un seul point d'accès avec trop d'appareils, dégradant ainsi les performances. |
