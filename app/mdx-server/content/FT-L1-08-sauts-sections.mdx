# FT-L1-08-sauts-sections
Voici une fiche technique sous forme de tableau, conçue pour des élèves de 7e année (niveau 1), qui aborde la notion de sauts de page et de sections dans Microsoft Word 2021 :

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Sauts de Page et Sections dans Microsoft Word 2021                                                                                                                                                                                                                                                                                                                                                                                         |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Insérer un saut de page <br/>• Créer une nouvelle section <br/>• Modifier les marges par section <br/>• Organiser efficacement un document long                                                                                                                                                                                                                |
| **Définitions clés**                          | **Saut de page** : Commande qui force le texte suivant à commencer sur une nouvelle page. <br/>**Section** : Partie d'un document qui peut avoir sa propre mise en forme, ses propres en-têtes et pieds de page. <br/>**Mise en page** : Ensemble des paramètres qui définissent l'apparence d'une page (marges, orientation, etc.).                                                                                                                                                                          |
| **Explication de la notion avec exemples**    | L'élève utilise les sauts de page pour commencer une nouvelle partie de son document sur une page fraîche, comme pour séparer les chapitres d'un livre. Les sections permettent d'appliquer des mises en page différentes dans un même document, par exemple avoir une page en orientation portrait et la suivante en paysage, ou des en-têtes différents selon les chapitres.                                                                                                                       |
| **Méthodologie ou étapes à suivre**           | 1. **Insérer un saut de page** : Placer le curseur où la nouvelle page doit commencer, aller dans l'onglet "Insertion" et cliquer sur "Saut de page".<br/>2. **Créer une section** : Placer le curseur où la nouvelle section doit commencer, aller dans l'onglet "Mise en page", cliquer sur "Sauts" et choisir "Page suivante" sous "Sauts de section".<br/>3. **Modifier les marges par section** : Double-cliquer dans la section, aller dans "Mise en page" > "Marges", choisir des marges différentes et cliquer sur "OK".<br/>4. **Personnaliser chaque section** : Modifier les en-têtes, pieds de page ou l'orientation de la page pour chaque section indépendamment. |
| **Illustration ou schéma explicatif**         | **Schéma des options de sauts dans Word** : <br/>• L'onglet **Insertion** contient l'option "Saut de page". <br/>• L'onglet **Mise en page** contient le menu "Sauts" avec les différents types de sauts de section. <br/>• Le menu **Marges** permet de personnaliser les marges pour chaque section.                                                                                   |
| **Exercice d'application ou question**        | **Exercice** : Demander à l'élève de créer un livre de recettes simple : <br/>1. Créer une page de titre <br/>2. Insérer un saut de page après la page de titre <br/>3. Créer trois sections différentes (entrées, plats, desserts) séparées par des sauts de section <br/>4. Personnaliser l'en-tête de chaque section avec le nom de la catégorie                                                                                                                    |
| **Mot-clé ou astuce mnémotechnique**          | **SAUTS** : <br/>**S**éparer les contenus – **A**jouter une nouvelle page – **U**tiliser les sections pour varier la mise en page – **T**erminer une partie – **S**tructurer le document                                                                                                                                                                                                                                                                                                                           |
