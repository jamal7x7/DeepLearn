# FT-L1-09-verification-orthographe
Voici une fiche technique sous forme de tableau, conçue pour des élèves de 7e année (niveau 1), qui aborde la notion de vérification orthographique dans Microsoft Word 2021 :

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Vérification Orthographique dans Microsoft Word 2021                                                                                                                                                                                                                                                                                                                                                                                         |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Comprendre le fonctionnement de la vérification automatique <br/>• Corriger les erreurs signalées <br/>• Lancer une vérification complète du document <br/>• Utiliser efficacement le correcteur orthographique                                                                                                                                                                                                                |
| **Définitions clés**                          | **Vérification orthographique** : Fonction qui détecte et signale les erreurs d'orthographe et de grammaire. <br/>**Soulignement rouge** : Indication d'une possible erreur d'orthographe. <br/>**Soulignement bleu** : Indication d'une possible erreur de grammaire ou de syntaxe.                                                                                                                                                                          |
| **Explication de la notion avec exemples**    | L'élève utilise le correcteur orthographique pour améliorer la qualité de ses textes. Par exemple, lorsqu'il écrit "ecole" sans accent, Word souligne le mot en rouge. En faisant un clic droit sur ce mot, l'élève peut voir la suggestion "école" et l'appliquer. Cela lui permet de rendre ses documents plus professionnels et sans fautes.                                                                                                                       |
| **Méthodologie ou étapes à suivre**           | 1. **Comprendre la vérification automatique** : Observer que Word souligne automatiquement les mots incorrects en rouge (orthographe) et en bleu (grammaire).<br/>2. **Corriger une erreur signalée** : Faire un clic droit sur le mot souligné, puis choisir la correction suggérée dans le menu contextuel.<br/>3. **Vérifier tout le document** : Aller dans l'onglet "Révision", cliquer sur "Orthographe et grammaire" pour lancer une vérification complète.<br/>4. **Gérer les suggestions** : Pour chaque erreur détectée, choisir "Ignorer" ou "Modifier" selon le contexte. |
| **Illustration ou schéma explicatif**         | **Schéma du correcteur orthographique** : <br/>• Les mots mal orthographiés sont **soulignés en rouge**. <br/>• Les erreurs grammaticales sont **soulignées en bleu**. <br/>• Le menu contextuel (clic droit) affiche les **suggestions de correction**. <br/>• L'onglet **Révision** contient l'outil de vérification complète.                                                                                   |
| **Exercice d'application ou question**        | **Exercice** : Demander à l'élève de corriger un texte contenant des erreurs : <br/>1. Copier un texte avec des fautes comme "Je vais a l'ecole tous les jour" <br/>2. Utiliser le correcteur orthographique pour identifier les erreurs <br/>3. Corriger toutes les fautes en utilisant les suggestions <br/>4. Vérifier si le correcteur a trouvé toutes les erreurs                                                                                                                    |
| **Mot-clé ou astuce mnémotechnique**          | **VERIF** : <br/>**V**oir les soulignements – **E**xaminer les suggestions – **R**ectifier les erreurs – **I**gnorer si nécessaire – **F**aire une vérification complète                                                                                                                                                                                                                                                                                                                           |
