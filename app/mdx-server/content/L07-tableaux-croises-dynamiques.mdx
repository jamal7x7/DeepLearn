# Tableaux Croisés Dynamiques dans Excel 2021

<ExerciseCard title="Synthétiser et analyser vos données">

Dans cette leçon, tu vas découvrir l'un des outils les plus puissants d'Excel : les tableaux croisés dynamiques. Ils te permettront de synthétiser, d'explorer et d'analyser rapidement de grandes quantités de données.

Voici ce que tu vas apprendre :
- Ce qu'est un tableau croisé dynamique et à quoi il sert
- Comment créer un tableau croisé dynamique simple
- Comment organiser les données en lignes, colonnes, valeurs et filtres
- Comment modifier la présentation et les calculs d'un tableau croisé dynamique

</ExerciseCard>

<Tip>
Si tu modifies la structure de ton tableau source (ajout de colonnes, changement de noms), n'oublie pas de mettre à jour la source de données du tableau croisé. Clique sur le tableau croisé, puis dans l'onglet "Analyser", clique sur "Modifier la source de données".
</Tip>

<ExerciseCard title="Exercice pratique: Analyse d'une enquête">

Crée un tableau croisé dynamique pour analyser les résultats d'une enquête auprès des élèves de ton école :

1. Crée un tableau de données avec les colonnes suivantes :
   - Classe (6e, 7e, 8e)
   - Genre (Fille, Garçon)
   - Sport préféré (Football, Basketball, Natation, Athlétisme, Autre)
   - Temps hebdomadaire consacré au sport (en heures)

2. Saisis au moins 20 enregistrements avec des données variées

3. Crée un tableau croisé dynamique sur une nouvelle feuille :
   - Place "Sport préféré" dans les Lignes
   - Place "Classe" dans les Colonnes
   - Place "Temps hebdomadaire" dans les Valeurs (en tant que Somme)

4. Modifie le calcul pour afficher la moyenne du temps hebdomadaire au lieu de la somme

5. Ajoute "Genre" comme filtre du rapport

6. Crée un second tableau croisé dynamique montrant :
   - "Genre" dans les Lignes
   - "Sport préféré" dans les Colonnes
   - Le nombre d'élèves dans les Valeurs

7. Crée un graphique croisé dynamique basé sur ton second tableau

Bonus : Groupe les classes 6e et 7e ensemble pour comparer avec la 8e.
