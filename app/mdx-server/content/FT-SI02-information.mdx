# FT-SI02 - Information

Cette fiche technique explique ce qu’est une information et comment elle est traitée par un ordinateur.

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                      |
|-----------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | La notion d’information                                                                                                                                                                                                        |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                              |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Définir ce qu’est une information <br/>• Distinguer donnée et information <br/>• Comprendre comment l’ordinateur traite l’information                                           |
| **Définitions clés**                          | **Donnée** : Élément brut (nombre, texte, image). <br/>**Information** : Donnée interprétée, qui a du sens.                                                                                                                     |
| **Explication de la notion avec exemples**    | La température "25" est une donnée. "Il fait chaud" est une information interprétée à partir de cette donnée.                                                                                                                   |
| **Méthodologie ou étapes à suivre**           | 1. Observer des données (nombres, textes) <br/>2. Discuter de leur signification <br/>3. Transformer des données en informations                                                                                               |
| **Illustration ou schéma explicatif**         | Schéma montrant la transformation de données en informations via un traitement.                                                                                                                                                |
| **Exercice d’application ou question**        | Donner une liste de données et demander à l’élève d’en déduire des informations.                                                                                                                                                |
| **Mot-clé ou astuce mnémotechnique**          | **DI** : <br/>**D**onnée <br/>→ **I**nformation                                                                                                                                                                                 |
