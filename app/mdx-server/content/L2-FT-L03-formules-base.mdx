# FT-L03-formules-base
Voici une fiche technique sous forme de tableau, conçue pour des élèves de 7e année (niveau 1), qui aborde la notion de formules de base dans Microsoft Excel 2021 :

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Formules de Base dans Excel 2021                                                                                                                                                                                                                                                                                                                                                                                                         |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Comprendre le principe des formules dans Excel <br/>• Créer des formules arithmétiques simples (+, -, *, /) <br/>• Utiliser les fonctions de base (SOMME, MOYENNE, MAX, MIN) <br/>• Comprendre la référence aux cellules dans les formules                                                                                                                                                                                                                |
| **Définitions clés**                          | **Formule** : Expression qui effectue un calcul sur des valeurs dans une feuille de calcul. <br/>**Fonction** : Formule prédéfinie qui effectue un calcul spécifique. <br/>**Référence de cellule** : Adresse d'une cellule (ex: A1) utilisée dans une formule. <br/>**Opérateur** : Symbole qui spécifie le type de calcul à effectuer (+, -, *, /).                                                                                                                                                                          |
| **Explication de la notion avec exemples**    | L'élève crée un tableau de notes d'élèves et utilise des formules pour calculer automatiquement les totaux et les moyennes. Par exemple, si les notes sont dans les cellules B2 à B6, il peut utiliser la formule =SOMME(B2:B6) pour calculer le total et =MOYENNE(B2:B6) pour calculer la moyenne. Il peut également utiliser des formules arithmétiques simples comme =B2+B3 pour additionner deux valeurs ou =B2*0.3+B3*0.7 pour calculer une moyenne pondérée.                                                                                                                       |
| **Méthodologie ou étapes à suivre**           | 1. **Commencer une formule** : Cliquer sur la cellule où doit apparaître le résultat et taper le signe égal (=).<br/>2. **Créer une formule arithmétique** : Après le signe égal, taper une expression mathématique en utilisant des références de cellules et des opérateurs (ex: =A1+A2).<br/>3. **Utiliser une fonction** : Taper le nom de la fonction suivi de parenthèses contenant les arguments (ex: =SOMME(A1:A5)).<br/>4. **Sélectionner des cellules** : Au lieu de taper les références, cliquer sur les cellules concernées pendant la création de la formule.<br/>5. **Valider la formule** : Appuyer sur Entrée pour confirmer et voir le résultat.<br/>6. **Copier une formule** : Utiliser la poignée de recopie (petit carré noir en bas à droite de la cellule) pour dupliquer la formule vers d'autres cellules. |
| **Illustration ou schéma explicatif**         | **Schéma des éléments d'une formule** : <br/>• Le **signe égal (=)** indique le début d'une formule. <br/>• Les **références de cellules** (A1, B2, etc.) pointent vers les données à utiliser. <br/>• Les **opérateurs** (+, -, *, /) définissent les calculs à effectuer. <br/>• Les **fonctions** (SOMME, MOYENNE, etc.) sont des formules prédéfinies. <br/>• Les **parenthèses** regroupent les arguments d'une fonction. <br/>• Les **deux-points (:)** définissent une plage de cellules (ex: A1:A5).                                                                                   |
| **Exercice d'application ou question**        | **Exercice** : Demander à l'élève de créer un tableau de budget mensuel contenant : <br/>1. Une liste de dépenses dans la colonne A (Loyer, Nourriture, Transport, Loisirs) <br/>2. Les montants correspondants dans la colonne B <br/>3. Dans la cellule B6, créer une formule qui calcule le total des dépenses <br/>4. Dans la cellule B7, créer une formule qui calcule la moyenne des dépenses <br/>5. Dans la cellule B8, créer une formule qui trouve la dépense la plus élevée <br/>6. Dans la cellule B9, créer une formule qui trouve la dépense la plus basse <br/>7. Bonus: Créer une formule qui calcule combien il reste d'un budget de 1000€ après toutes les dépenses                                                                                                                    |
| **Mot-clé ou astuce mnémotechnique**          | **FORMULE** : <br/>**F**onction ou calcul – **O**pérations mathématiques – **R**éférences de cellules – **M**anipulation des données – **U**tilisation du signe égal – **L**ier les cellules entre elles – **E**xécution automatique des calculs                                                                                                                                                                                                                                                                                                                           |
