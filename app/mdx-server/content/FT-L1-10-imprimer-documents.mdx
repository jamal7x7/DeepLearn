# FT-L1-10-imprimer-documents
Voici une fiche technique sous forme de tableau, conçue pour des élèves de 7e année (niveau 1), qui aborde la notion d'impression de documents dans Microsoft Word 2021 :

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Imprimer des Documents dans Microsoft Word 2021                                                                                                                                                                                                                                                                                                                                                                                         |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Prévisualiser un document avant impression <br/>• Choisir une imprimante <br/>• Configurer les options d'impression <br/>• Imprimer un document complet ou partiel                                                                                                                                                                                                                |
| **Définitions clés**                          | **Aperçu avant impression** : Visualisation du document tel qu'il apparaîtra une fois imprimé. <br/>**Options d'impression** : Paramètres configurables comme le nombre de copies, les pages à imprimer, l'orientation, etc. <br/>**Orientation** : Disposition de la page (portrait ou paysage).                                                                                                                                                                          |
| **Explication de la notion avec exemples**    | L'élève utilise la fonction d'impression pour obtenir une version papier de son document. Par exemple, avant d'imprimer un exposé, il prévisualise le document pour s'assurer que la mise en page est correcte, puis configure les options d'impression selon ses besoins (nombre de copies, pages spécifiques, etc.) avant de lancer l'impression.                                                                                                                       |
| **Méthodologie ou étapes à suivre**           | 1. **Prévisualiser** : Cliquer sur "Fichier" puis "Imprimer" pour voir l'aperçu du document à droite de l'écran.<br/>2. **Choisir l'imprimante** : Dans la section "Imprimante", sélectionner l'imprimante souhaitée dans la liste déroulante.<br/>3. **Configurer les options** : Choisir le nombre de copies, sélectionner les pages à imprimer (toutes ou certaines pages), choisir l'orientation (portrait ou paysage).<br/>4. **Imprimer** : Après avoir vérifié que tous les paramètres sont corrects, cliquer sur le bouton "Imprimer". |
| **Illustration ou schéma explicatif**         | **Schéma des options d'impression** : <br/>• Le menu **Fichier > Imprimer** affiche l'aperçu et les options. <br/>• La section **Imprimante** permet de choisir le périphérique d'impression. <br/>• Les options de **Pages** permettent de sélectionner les pages à imprimer. <br/>• L'option **Orientation** permet de choisir entre portrait et paysage.                                                                                   |
| **Exercice d'application ou question**        | **Exercice** : Demander à l'élève de préparer un document pour l'impression : <br/>1. Créer une invitation simple pour un événement <br/>2. Prévisualiser le document avant impression <br/>3. Configurer l'impression en mode paysage <br/>4. Ajuster les marges pour centrer le texte sur la page                                                                                                                    |
| **Mot-clé ou astuce mnémotechnique**          | **IMPRIM** : <br/>**I**nspecter l'aperçu – **M**odifier les options si nécessaire – **P**réciser le nombre de copies – **R**égler l'orientation – **I**ndiquer les pages à imprimer – **M**ettre en marche l'impression                                                                                                                                                                                                                                                                                                                           |
