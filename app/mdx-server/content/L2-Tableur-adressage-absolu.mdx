### III- L'adressage absolu dans Excel

##### 1- Qu'est-ce que l'adressage absolu?

Dans Excel, lorsque vous créez une formule qui fait référence à une cellule (par exemple =A1+B1), cette référence est par défaut **relative**. Cela signifie que si vous copiez cette formule dans une autre cellule, les références seront automatiquement ajustées.

L'**adressage absolu** permet de "fixer" une référence pour qu'elle ne change pas lors de la copie d'une formule. On utilise le symbole $ pour cela.

##### 2- Les différents types d'adressage

| Type d'adressage | Notation | Comportement lors de la copie |
|-----------------|----------|-------------------------------|
| **Adressage relatif** | `A1` | Change en ligne et en colonne |
| **Adressage absolu** | `$A$1` | Ne change ni en ligne ni en colonne |
| **Adressage mixte (colonne fixe)** | `$A1` | Colonne fixe, ligne relative |
| **Adressage mixte (ligne fixe)** | `A$1` | Colonne relative, ligne fixe |

## Pourquoi utiliser l'adressage absolu?

L'adressage absolu est particulièrement utile dans les situations suivantes:

- Référencer une valeur constante (comme un taux de TVA)
- Créer des tableaux de calcul avec des multiplicateurs fixes
- Appliquer la même formule à plusieurs lignes ou colonnes
- Calculer des pourcentages par rapport à une valeur totale

## Exemple pratique: Calcul de TVA

Imaginez que vous avez une liste de prix HT (Hors Taxes) et que vous souhaitez calculer les prix TTC (Toutes Taxes Comprises) avec un taux de TVA de 20%.

<ExerciseCard title="Exemple: Calcul de TVA">

1. Dans la cellule A1, écrivez "Produit"
2. Dans la cellule B1, écrivez "Prix HT"
3. Dans la cellule C1, écrivez "Prix TTC"
4. Dans la cellule E1, écrivez "Taux TVA"
5. Dans la cellule E2, entrez la valeur `0.2` (20%)

Remplissez ensuite quelques produits avec leurs prix HT:

| Cellule | Produit | Prix HT |
|---------|---------|--------|
| A2, B2 | Produit 1 | 100 |
| A3, B3 | Produit 2 | 150 |
| A4, B4 | Produit 3 | 200 |

Pour calculer le prix TTC du premier produit, dans la cellule C2, entrez la formule:
```
=B2*(1+E2)
```

Si vous copiez cette formule vers le bas (C3, C4), la référence à B2 s'ajustera (B3, B4), mais la référence à E2 changera aussi (E3, E4) - ce qui n'est pas ce que nous voulons!

Utilisez plutôt l'adressage absolu pour le taux de TVA:
```
=B2*(1+$E$2)
```

Maintenant, vous pouvez copier cette formule vers le bas et le taux de TVA restera toujours fixé à la cellule E2.

## Exercice pratique: Tableau de conversion de devises

<ExerciseCard title="Exercice: Tableau de conversion de devises">

Créez un tableau de conversion entre l'Euro et d'autres devises:

1. Configurez votre tableau comme suit:
   - A1: "Devise"
   - B1: "Taux de change"
   - C1: "Montant en Euros"
   - D1: "Montant converti"

2. Entrez les devises et leurs taux de change:

   | Cellule | Devise | Taux de change |
   |---------|--------|---------------|
   | A2, B2 | USD (Dollar américain) | 1.08 |
   | A3, B3 | GBP (Livre sterling) | 0.85 |
   | A4, B4 | JPY (Yen japonais) | 160.5 |

3. Dans la cellule C2, entrez un montant en euros (par exemple 100)

4. Dans la cellule D2, créez une formule pour convertir les euros en dollars:
   ```
   =C2*B2
   ```

5. Copiez cette formule dans les cellules D3 et D4. Que se passe-t-il?

6. Modifiez maintenant la formule en D2 pour utiliser l'adressage absolu pour le montant en euros:
   ```
   =$C$2*B2
   ```

7. Copiez à nouveau cette formule dans les cellules D3 et D4.

8. Changez la valeur en C2 et observez comment tous les montants convertis sont mis à jour automatiquement!

## Défi: Calcul de pourcentages

<ExerciseCard title="Défi: Calcul de pourcentages par rapport à un total">

Créez un tableau qui calcule le pourcentage que représente chaque valeur par rapport à un total:

1. Configurez votre tableau:
   ```
   A1: Catégorie
   B1: Montant
   C1: Pourcentage du total
   ```

2. Entrez quelques catégories avec leurs montants:

   | Cellule | Catégorie | Montant |
   |---------|-----------|--------|
   | A2, B2 | Alimentation | 350 |
   | A3, B3 | Logement | 800 |
   | A4, B4 | Transport | 150 |
   | A5, B5 | Loisirs | 200 |

3. Dans la cellule B6, calculez le total avec la formule:
   ```
   =SOMME(B2:B5)
   ```

4. Dans la cellule C2, créez une formule pour calculer le pourcentage:
   ```
   =B2/B6
   ```

5. Copiez cette formule dans les cellules C3, C4 et C5. Les résultats sont-ils corrects?

6. Modifiez la formule en C2 en utilisant l'adressage absolu:
   ```
   =B2/$B$6
   ```

7. Copiez à nouveau cette formule et vérifiez que les pourcentages sont maintenant calculés correctement.

8. Formatez les cellules C2:C5 en pourcentage pour une meilleure lisibilité.

</ExerciseCard>

## Exercice avancé: Tableau de données avec filtres

<ExerciseCard title="Exercice: Tableau de données avec filtres">

Dans cet exercice, vous allez créer un tableau de données plus complexe qui utilise l'adressage absolu pour maintenir des références constantes lors du filtrage.

1. Configurez votre tableau comme suit:
   ```
   A1: Région
   B1: Vendeur
   C1: Produit
   D1: Quantité
   E1: Prix unitaire
   F1: Total
   G1: % du CA global
   ```

2. Entrez les données suivantes:

   | Cellule | Région | Vendeur | Produit | Quantité | Prix unitaire |
   |---------|--------|---------|---------|----------|---------------|
   | A2-E2 | Nord | Martin | Ordinateur | 5 | 800 |
   | A3-E3 | Sud | Dupont | Imprimante | 10 | 200 |
   | A4-E4 | Est | Durand | Ordinateur | 3 | 850 |
   | A5-E5 | Ouest | Martin | Écran | 8 | 150 |
   | A6-E6 | Nord | Dupont | Ordinateur | 4 | 820 |
   | A7-E7 | Sud | Durand | Écran | 12 | 160 |

3. Dans la cellule F2, créez une formule pour calculer le total (Quantité × Prix unitaire):
   ```
   =D2*E2
   ```

4. Copiez cette formule dans les cellules F3:F7.

5. Dans la cellule H2, calculez le CA global (somme de tous les totaux):
   ```
   =SOMME(F2:F7)
   ```

6. Dans la cellule G2, créez une formule pour calculer le pourcentage que représente chaque vente par rapport au CA global:
   ```
   =F2/H2
   ```

7. Si vous copiez cette formule dans les cellules G3:G7, que se passe-t-il? Pourquoi?

8. Modifiez la formule en G2 en utilisant l'adressage absolu:
   ```
   =F2/$H$2
   ```

9. Copiez cette formule dans les cellules G3:G7 et vérifiez que les pourcentages sont corrects.

10. Formatez les cellules G2:G7 en pourcentage.

11. Utilisez les filtres automatiques (Données > Filtre) sur votre tableau pour:
    - Afficher uniquement les ventes d'ordinateurs
    - Afficher uniquement les ventes du vendeur Martin

12. Observez que même avec les filtres activés, les pourcentages restent corrects grâce à l'adressage absolu!

## Conclusion

L'adressage absolu est une compétence fondamentale dans Excel qui vous permet de créer des formules plus flexibles et réutilisables. En maîtrisant cette technique, vous pourrez:

- Créer des tableaux de calcul complexes
- Éviter les erreurs lors de la copie de formules
- Gagner du temps en réutilisant les mêmes formules
- Créer des modèles de calcul robustes
- Travailler efficacement avec des tableaux de données et des tableaux croisés dynamiques
- Maintenir l'intégrité de vos calculs même lors du filtrage ou de la réorganisation des données

Pratiquez régulièrement avec différents scénarios pour devenir à l'aise avec cette fonctionnalité essentielle d'Excel! */}
