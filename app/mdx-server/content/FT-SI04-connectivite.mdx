# Fiche Technique : La Connectivité

| **Titre du thème**                            | La Connectivité                                                                                                                                                                                                                                                                                                                                                                                                         |
| **Niveau concerné**                           | Intermédiaire                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Définir ce qu'est la connectivité dans un contexte informatique <br/>• Identifier les différents types de connexions et de réseaux <br/>• Comprendre les protocoles de communication essentiels <br/>• Expliquer les enjeux de sécurité liés à la connectivité                                                                                                                                                                                                                |
| **Définitions clés**                          | **Connectivité** : Capacité des systèmes informatiques à se connecter entre eux et à échanger des informations. <br/>**Réseau** : Ensemble d'équipements informatiques reliés entre eux pour échanger des données. <br/>**Protocole** : Ensemble de règles qui définissent comment les données sont transmises entre les appareils. <br/>**Bande passante** : Quantité de données pouvant être transmises dans un laps de temps donné.                                                                                                                                                                                          |
| **Explication de la notion avec exemples**    | La connectivité est l'ensemble des technologies qui permettent aux systèmes informatiques de communiquer entre eux. Elle est fondamentale dans notre monde numérique où les appareils doivent constamment échanger des informations. On distingue plusieurs types de connectivité : filaire (Ethernet, USB, HDMI) et sans fil (Wi-Fi, Bluetooth, 4G/5G). Chaque type a ses avantages et inconvénients en termes de vitesse, de portée, de fiabilité et de sécurité. Par exemple, une connexion Ethernet offre généralement une meilleure stabilité et des débits plus élevés qu'une connexion Wi-Fi, mais elle nécessite un câble physique. Les réseaux peuvent être locaux (LAN) comme un réseau domestique, ou étendus (WAN) comme Internet. La communication entre les appareils est régie par des protocoles comme TCP/IP, qui est la base d'Internet. Ces protocoles définissent comment les données sont formatées, adressées, transmises, reçues et accusées de réception.                                                                                                                       |
| **Méthodologie ou étapes à suivre**           | 1. **Analyse des besoins** : Déterminer les exigences en matière de connectivité (vitesse, mobilité, sécurité).<br/>2. **Choix des technologies** : Sélectionner les types de connexions adaptés aux besoins.<br/>3. **Configuration du réseau** : Mettre en place l'infrastructure réseau (routeurs, commutateurs, etc.).<br/>4. **Paramétrage des protocoles** : Configurer les protocoles de communication.<br/>5. **Sécurisation** : Mettre en place des mesures de protection (pare-feu, chiffrement, etc.).<br/>6. **Test** : Vérifier que la connectivité fonctionne correctement.<br/>7. **Maintenance** : Surveiller et mettre à jour régulièrement le système. |

| **Conseils pratiques ou astuces**             | • Privilégiez les connexions filaires pour les appareils fixes nécessitant une connexion stable.<br/>• Positionnez correctement vos routeurs Wi-Fi pour maximiser la couverture.<br/>• Utilisez des réseaux privés virtuels (VPN) pour sécuriser vos connexions sur les réseaux publics.<br/>• Mettez régulièrement à jour le firmware de vos équipements réseau.<br/>• Segmentez votre réseau pour isoler les appareils sensibles des autres. |
| **Erreurs courantes à éviter**                | • Négliger la sécurité des connexions sans fil.<br/>• Sous-dimensionner la bande passante nécessaire.<br/>• Ignorer les interférences potentielles pour les connexions sans fil.<br/>• Ne pas mettre à jour les équipements réseau.<br/>• Utiliser des protocoles obsolètes ou non sécurisés. |
