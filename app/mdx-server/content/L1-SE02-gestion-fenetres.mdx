# Gestion des Fenêtres

<ExerciseCard title="Maîtriser la gestion des fenêtres">

Dans cette leçon, tu vas apprendre à manipuler efficacement les fenêtres dans un système d'exploitation graphique, ce qui te permettra de travailler plus rapidement et d'organiser ton espace de travail numérique.

Voici ce que tu vas apprendre :
- Ce qu'est une fenêtre et ses composants
- Comment manipuler les fenêtres (ouvrir, fermer, déplacer, redimensionner)
- Comment organiser plusieurs fenêtres à l'écran
- Les raccourcis clavier pour une gestion efficace des fenêtres

</ExerciseCard>

      <div>
        <h4>Ouvrir une fenêtre</h4>
        <p>Pour ouvrir une fenêtre, tu peux :</p>
        <ul>
          <li>Cliquer sur une icône d'application dans le menu Démarrer, le Dock ou le Launchpad</li>
          <li>Double-cliquer sur une icône de document ou d'application sur le bureau</li>
          <li>Utiliser un raccourci clavier associé à une application</li>
        </ul>
        
        <h4>Déplacer une fenêtre</h4>
        <p>Pour déplacer une fenêtre :</p>
        <ol>
          <li>Place le curseur sur la barre de titre</li>
          <li>Clique et maintiens le bouton gauche de la souris enfoncé</li>
          <li>Fais glisser la fenêtre vers l'emplacement souhaité</li>
          <li>Relâche le bouton de la souris</li>
        </ol>
        
        <h4>Redimensionner une fenêtre</h4>
        <p>Pour redimensionner une fenêtre :</p>
        <ol>
          <li>Place le curseur sur un bord ou un coin de la fenêtre (le curseur change de forme)</li>
          <li>Clique et maintiens le bouton gauche de la souris enfoncé</li>
          <li>Fais glisser pour agrandir ou réduire la fenêtre</li>
          <li>Relâche le bouton de la souris</li>
        </ol>
        
        <h4>Maximiser une fenêtre</h4>
        <p>Pour maximiser une fenêtre (l'agrandir pour qu'elle occupe tout l'écran) :</p>
        <ul>
          <li>Clique sur le bouton de maximisation [□] dans la barre de titre</li>
          <li>Double-clique sur la barre de titre</li>
          <li>Utilise le raccourci clavier (Win+Flèche haut sur Windows, Ctrl+Flèche verte sur macOS)</li>
        </ul>
        
        <h4>Minimiser une fenêtre</h4>
        <p>Pour minimiser une fenêtre (la réduire dans la barre des tâches ou le Dock) :</p>
        <ul>
          <li>Clique sur le bouton de minimisation [_] dans la barre de titre</li>
          <li>Utilise le raccourci clavier (Win+Flèche bas sur Windows, Cmd+M sur macOS)</li>
        </ul>
        
        <h4>Fermer une fenêtre</h4>
        <p>Pour fermer une fenêtre :</p>
        <ul>
          <li>Clique sur le bouton de fermeture [X] dans la barre de titre</li>
          <li>Utilise le raccourci clavier (Alt+F4 sur Windows, Cmd+W sur macOS)</li>

<ExerciseCard title="Exercice pratique : Maîtriser les fenêtres">

Mets en pratique ce que tu as appris avec ces exercices :

### Exercice 1 : Manipulation de base

1. Ouvre trois applications différentes (par exemple, un navigateur web, un traitement de texte et une calculatrice).
2. Pratique le déplacement de chaque fenêtre à différents endroits de l'écran.
3. Redimensionne chaque fenêtre pour qu'elle occupe environ un quart de l'écran.
4. Maximise puis restaure chaque fenêtre.
5. Minimise puis restaure chaque fenêtre.

### Exercice 2 : Organisation des fenêtres

1. Organise les trois fenêtres ouvertes pour qu'elles soient toutes visibles simultanément à l'écran.
2. Utilise la fonction d'ancrage (snap) pour placer deux fenêtres côte à côte.
3. Pratique le basculement entre les fenêtres en utilisant Alt+Tab (ou Cmd+Tab).
4. Si ton système le permet, crée un nouveau bureau virtuel et déplace une des applications vers ce bureau.

### Exercice 3 : Raccourcis clavier

1. Ferme et rouvre une application en utilisant uniquement le clavier.
2. Maximise et restaure une fenêtre en utilisant uniquement le clavier.
3. Bascule entre toutes tes applications ouvertes en utilisant uniquement le clavier.
4. Affiche le bureau (minimise toutes les fenêtres) puis restaure-les, toujours avec le clavier.

### Exercice 4 : Défi avancé

1. Chronomètre-toi pour voir combien de temps il te faut pour ouvrir trois applications et les organiser en mosaïque à l'écran en utilisant la souris.
2. Refais le même exercice en utilisant uniquement les raccourcis clavier.
3. Compare les deux temps pour voir à quel point les raccourcis peuvent te faire gagner du temps !

</ExerciseCard>
