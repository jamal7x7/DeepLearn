# FT-SI04 - Fonctionnement d’un système informatique

Cette fiche technique explique comment un système informatique fonctionne dans son ensemble.

| **Rubrique**                                  | **Contenu**                                                                                                                                                                                                                      |
|-----------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Titre du thème**                            | Fonctionnement global d’un système informatique                                                                                                                                                                                 |
| **Niveau concerné**                           | 7e                                                                                                                                                                                                                              |
| **Objectif pédagogique**                      | À la fin de cette leçon, l'élève saura : <br/>• Comprendre le cycle Entrée-Traitement-Sortie <br/>• Identifier les étapes du traitement de l’information <br/>• Relier matériel et logiciel                                    |
| **Définitions clés**                          | **Entrée** : Données saisies (clavier, souris). <br/>**Traitement** : Calculs effectués par le processeur. <br/>**Sortie** : Résultat affiché (écran, impression).                                                              |
| **Explication de la notion avec exemples**    | Taper un texte (entrée), l’ordinateur l’affiche et le sauvegarde (traitement), puis l’imprime (sortie).                                                                                                                        |
| **Méthodologie ou étapes à suivre**           | 1. Identifier les périphériques d’entrée et sortie <br/>2. Suivre le cheminement d’une donnée <br/>3. Relier chaque étape au matériel concerné                                                                                  |
| **Illustration ou schéma explicatif**         | Schéma du cycle **Entrée → Traitement → Sortie** avec exemples.                                                                                                                                                                |
| **Exercice d’application ou question**        | Demander à l’élève de décrire le cycle pour une action (ex: imprimer une image).                                                                                                                                               |
| **Mot-clé ou astuce mnémotechnique**          | **ETS** : <br/>**E**ntrée <br/>**T**raitement <br/>**S**ortie                                                                                                                                                                  |
